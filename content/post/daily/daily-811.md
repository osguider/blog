stremio-web 是一个现代化的媒体中心，旨在为用户提供一站式的视频娱乐解决方案。

- 支持通过易于安装的插件发现、观看和管理视频内容

## [glide-browser/glide](https://github.com/glide-browser/glide)

{{< shields path="github/stars/glide-browser/glide" alt="Github Repo Stars" >}} {{< shields path="github/license/glide-browser/glide" alt="License: `MPL-2.0`" >}} {{< shields path="github/languages/top/glide-browser/glide" alt="Language: `Unknown`" >}}

{{< github-opengraph user="glide-browser" repo="glide" alt="cover" >}}

glide 是一个基于 Firefox 构建的可扩展且以键盘操作为核心的网页浏览器。

- 支持仅通过键盘进行网页导航，提升操作效率
- 采用模态化按键映射设计，方便用户自定义快捷键
- 使用 TypeScript 进行配置，提高配置的灵活性和安全性
- 支持 Web 扩展 API，增强浏览器功能扩展能力
- 内置模糊标签管理器，通过快捷组合键快速切换标签页
- 提供站点特定设置和按键映射，实现个性化使用体验
  
## [foundryzero/binder-trace](https://github.com/foundryzero/binder-trace)

{{< shields path="github/stars/foundryzero/binder-trace" alt="Github Repo Stars" >}} {{< shields path="github/license/foundryzero/binder-trace" alt="License: `MIT`" >}} {{< shields path="github/languages/top/foundryzero/binder-trace" alt="Language: `Unknown`" >}}

![demo-picture-of-binder-trace](https://static.osguider.com/subject/github/foundryzero/binder-trace/c4413871252bdd979c2d8d09acb4e20e.png)

binder-trace 是一个用于拦截和解析 Android Binder 消息的工具，类似于 "Wireshark for Binder"。

- 支持与根权限的 Android 设备或模拟器连接
- 提供命令行界面以便用户选择要附加的设备和进程
- 可通过配置文件过滤特定接口、方法和类型
- 实时记录并显示 Binder 事务，支持暂停/恢复功能
- 简单易用，可通过 pip 安装并快速上手

## [dissorial/doc-chatbot](https://github.com/dissorial/doc-chatbot)

{{< shields path="github/stars/dissorial/doc-chatbot" alt="Github Repo Stars" >}} {{< shields path="github/license/dissorial/doc-chatbot" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/dissorial/doc-chatbot" alt="Language: `Unknown`" >}}

![demo-picture-of-doc-chatbot](https://static.osguider.com/subject/github/dissorial/doc-chatbot/e43c7752746a862ac4ef1c6e54317487.png)

doc-chatbot 是一个基于 GPT 的文档聊天机器人，支持多个文件、主题和聊天窗口，以及聊天历史记录的管理。

- 创建多个主题进行对话
- 为每个主题存储任意数量的文件
- 为每个主题创建任意数量的聊天窗口
- 上传文件并将其转换为嵌入，存储在 Pinecone 中，并可通过浏览器删除命名空间
- 使用本地存储自动保存和检索所有聊天记录
- 支持 .pdf、.docx 和 .txt 文件格式
- 集成 LangChain 和 Pinecone 以增强功能
