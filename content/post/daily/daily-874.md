Lean 是一个面向量化交易者的事件驱动型算法交易引擎，支持多市场的策略回测与实盘交易。

- 采用模块化设计，组件可插拔且高度可定制，内置主流插件模型
- 支持丰富的替代数据源和实时交易功能，实现从策略开发到部署的一体化流程
- 提供跨平台命令行工具（LEAN CLI），简化项目管理、回测、优化及实盘操作
- 支持本地与云端混合开发环境，兼容多种 IDE 并具备完整调试和自动补全能力
  
## [BloopAI/vibe-kanban](https://github.com/BloopAI/vibe-kanban)

{{< shields path="github/stars/BloopAI/vibe-kanban" alt="Github Repo Stars" >}} {{< shields path="github/license/BloopAI/vibe-kanban" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/BloopAI/vibe-kanban" alt="Language: `Unknown`" >}}

{{< github-opengraph user="BloopAI" repo="vibe-kanban" alt="cover" >}}

vibe-kanban 是一个用于提升多种 AI 编码代理协同效率的任务管理和执行编排工具。

- 支持在 Claude Code、Gemini CLI、Codex、Amp 等多个编码代理间灵活切换
- 可并行或顺序编排多个编码代理的任务执行流程
- 提供快速代码审查与开发服务器启动功能，简化开发过程监控
- 实时跟踪各个编码代理正在处理的任务状态，实现集中管理
- 集中配置多种编码代理的 MCP 配置文件，统一维护环境设置
- 支持通过 SSH 远程打开项目，方便在远程服务器上运行和操作
  
## [ventoy/Ventoy](https://github.com/ventoy/Ventoy)

{{< shields path="github/stars/ventoy/Ventoy" alt="Github Repo Stars" >}} {{< shields path="github/license/ventoy/Ventoy" alt="License: `GPL-3.0`" >}} {{< shields path="github/languages/top/ventoy/Ventoy" alt="Language: `Unknown`" >}}

![demo-picture-of-Ventoy](https://osguider.oss-cn-guangzhou.aliyuncs.com/subject/cff845a8daa02e012817d1c2d7cae233.png)

Ventoy 是一款开源工具，用于创建支持多种镜像文件格式的可启动 USB 驱动器，无需反复格式化，直接复制镜像即可启动。

- 支持 ISO、WIM、IMG、VHD(x)、EFI 等多种镜像文件格式
- 允许同时复制多个镜像文件，并通过启动菜单选择引导项
- 兼容 x86 Legacy BIOS 及多种 UEFI 架构（IA32/x86_64/ARM64/MIPS64EL）
- 同时支持 MBR 和 GPT 分区样式，无差异使用体验
- 广泛兼容主流操作系统，包括 Windows 系列、多达 1300+ 经过测试的 Linux 发行版及其他系统环境
  
## [nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)

{{< shields path="github/stars/nextlevelbuilder/ui-ux-pro-max-skill" alt="Github Repo Stars" >}} {{< shields path="github/license/nextlevelbuilder/ui-ux-pro-max-skill" alt="License: `MIT`" >}} {{< shields path="github/languages/top/nextlevelbuilder/ui-ux-pro-max-skill" alt="Language: `Unknown`" >}}

{{< github-opengraph user="nextlevelbuilder" repo="ui-ux-pro-max-skill" alt="cover" >}}

ui-ux-pro-max-skill 是一个为多平台专业 UI/UX 设计提供智能辅助的 AI 技能库。

- 包含丰富的设计资源：57 种 UI 风格、95 套行业色彩调色板、56 组字体搭配和 24 种图表类型，覆盖广泛视觉元素
- 支持多技术栈最佳实践：涵盖 React、Next.js、Vue 等 10 大主流前端框架及移动开发环境
- 集成详尽 UX 指南：98 条用户体验规范，包括最佳实践、防反模式及无障碍规则
- 可与多款 AI 编码助手无缝协作，如 Claude Code、Codex 和 GitHub Copilot，提升设计效率和一致性
- 提供命令行工具便捷安装与管理，同时支持手动集成到不同项目结构中
  
## [spring-projects/spring-boot](https://github.com/spring-projects/spring-boot)

{{< shields path="github/stars/spring-projects/spring-boot" alt="Github Repo Stars" >}} {{< shields path="github/license/spring-projects/spring-boot" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/spring-projects/spring-boot" alt="Language: `Unknown`" >}}

{{< github-opengraph user="spring-projects" repo="spring-boot" alt="cover" >}}

spring-boot 是一个帮助开发者以最简便方式构建基于 Spring 框架的生产级应用和服务的工具。

- 提供开箱即用的快速启动体验，支持独立运行的 Java 应用（java -jar）及传统 WAR 部署
- 采用约定优于配置原则，减少繁琐设置，无需代码生成或 XML 配置
- 集成常见非功能性特性，如内嵌服务器、安全、指标监控、健康检查和外部化配置
- 附带命令行工具，可执行 Spring 脚本，提高开发效率
