## [wmjordan/PDFPatcher](https://github.com/wmjordan/PDFPatcher)

{{< shields path="github/stars/wmjordan/PDFPatcher" alt="Github Repo Stars" >}} {{< shields path="github/license/wmjordan/PDFPatcher" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/wmjordan/PDFPatcher" alt="Language: `Unknown`" >}}

{{< github-opengraph user="wmjordan" repo="PDFPatcher" alt="cover" >}}

PDFPatcher 是一款功能全面的 PDF 处理工具箱。

- 支持编辑书签（批量修改属性、查找替换、自动生成）、剪裁和旋转页面。
- 可解除复制及打印限制，清理隐藏垃圾数据，重新压缩图片，提高文档质量。
- 提供合并与拆分 PDF 文件功能，同时保留或添加自定义书签。
- 支持高速无损导出图片，将 PDF 页面转换为图片，并提取或删除指定页面。
- 利用微软 Office 图像识别引擎进行文字识别，实现目录页转书签等智能操作。
- 替换字体并嵌入字库，解决乱码问题，提高跨设备阅读兼容性。
- 以树视图分析和编辑文档结构，可导出 XML 文件便于调试和研究。
- 完全免费且无广告，无弹窗，不收集用户隐私。
  
## [HuLaSpark/HuLa](https://github.com/HuLaSpark/HuLa)

{{< shields path="github/stars/HuLaSpark/HuLa" alt="Github Repo Stars" >}} {{< shields path="github/license/HuLaSpark/HuLa" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/HuLaSpark/HuLa" alt="Language: `Unknown`" >}}

{{< github-opengraph user="HuLaSpark" repo="HuLa" alt="cover" >}}

HuLa 是一款基于 Rust 和 Vue3 技术栈，利用 Tauri、Vite 7 和 TypeScript 构建的跨平台即时通讯桌面应用，支持 Windows、macOS、Linux 以及移动端 iOS 和 Android 系统。

- 跨平台支持广泛，覆盖主流桌面和移动操作系统
- 高性能轻量级架构，结合 Tauri 提供安全且资源占用低的运行环境
- 丰富的消息功能，包括一对一私聊、群组聊天、多样化消息类型及撤回等
- 完善的社交管理体系，如好友添加删除、群组管理及在线状态显示
- 支持多设备登录与二维码扫码登录，提高使用便捷性和灵活性
- 界面现代美观，可切换深浅色主题与皮肤，自带截图和图片查看器等实用工具
- 集成 AI 聊天助手插件，实现智能辅助交流体验
- 自动更新机制保证软件持续迭代优化
  
## [KovenYu/WonderJourney](https://github.com/KovenYu/WonderJourney)

{{< shields path="github/stars/KovenYu/WonderJourney" alt="Github Repo Stars" >}} {{< shields path="github/license/KovenYu/WonderJourney" alt="License: `MIT`" >}} {{< shields path="github/languages/top/KovenYu/WonderJourney" alt="Language: `Unknown`" >}}

![demo-picture-of-WonderJourney](https://static.osguider.com/subject/github/KovenYu/WonderJourney/09644a4ffceabd6ae28b551365a27ce3.png)

WonderJourney 是一个用于从任何地方生成场景视频的开源项目。

- 支持使用 CUDA 兼容 GPU，要求 24GB GPU 内存。
- 基于 Pytorch3D 实现渲染功能。
- 提供示例配置文件，便于用户快速上手。
- 可通过 GPT-4 自动生成场景描述，提高创作效率。
- 用户可以自定义图像和内容提示，以创建独特的视频效果。

{{< shields path="github/stars/MichaelYuhe/ai-group-tabs" alt="Github Repo Stars" >}} {{< shields path="github/license/MichaelYuhe/ai-group-tabs" alt="License: `MIT`" >}} {{< shields path="github/languages/top/MichaelYuhe/ai-group-tabs" alt="Language: `Unknown`" >}}

![demo-picture-of-ai-group-tabs](https://static.osguider.com/subject/github/MichaelYuhe/ai-group-tabs/7eced204383716f6dbb1d41d15b01aaf.png)

ai-group-tabs 是一个利用人工智能组织和分组 Chrome 标签页的工具。

- 默认分类自动分组标签页
- 支持在弹出窗口中填写 OpenAI API 密钥并保存到 Chrome 存储
- 可自定义分类设置
- 自动对新打开的标签进行分组
- 提供 Chrome 商店下载链接，方便用户获取和使用
