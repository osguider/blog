## [emberjs/ember.js](https://github.com/emberjs/ember.js)

{{< shields path="github/stars/emberjs/ember.js" alt="Github Repo Stars" >}} {{< shields path="github/license/emberjs/ember.js" alt="License: `MIT`" >}} {{< shields path="github/languages/top/emberjs/ember.js" alt="Language: `Unknown`" >}}

![demo-picture-of-ember.js](https://static.osguider.com/subject/github/emberjs/ember.js/e765e2eb4eb3a8758a4044a9cbe017fe.png)

Ember.js 是一个用于创建雄心勃勃的 Web 应用程序的 JavaScript 框架。

- 提供友好的社区支持，随时获得帮助
- 具有持久的应用基础，早期版本至今仍在使用
- 定期发布 LTS 版本和安全修复，确保可靠性与安全性
- 支持现代 JavaScript 特性，如类、装饰器和生成器
- 拥有优质文档，为每个 Ember 版本提供详细指导
- 组件以有效且语义化的 HTML 为基础，可按需添加功能
- 路由功能尊重 URL，并增加额外功能如渲染模板和加载数据模型
- 默认集成强大的数据管理工具 Ember Data，也支持其他选项
- 灵活适配任何后端栈，通过适配器和序列化程序实现兼容
- 自动跟踪反应模型简化更新决策过程
- 强大的默认配置使得零配置应用成为可能，但也提供了更多选项
- 高质量插件生态系统，让增强应用变得更加简单

- 云中立性：在开发、本地及云环境中提供一致体验。
- 渐进式采用：可以将 Radius 集成到现有工作流程和基础架构代码模板目录中。
  
## [baidu/dperf](https://github.com/baidu/dperf)

{{< shields path="github/stars/baidu/dperf" alt="Github Repo Stars" >}} {{< shields path="github/license/baidu/dperf" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/baidu/dperf" alt="Language: `Unknown`" >}}

{{< github-opengraph user="baidu" repo="dperf" alt="cover" >}}

dperf 是一款基于 DPDK 的 100Gbps 网络性能和负载测试软件。

- 高性能：能够在单个 x86 服务器上生成巨大的流量，支持数千万 HTTP 每秒、数百 Gbps 吞吐量和数十亿并发连接。
- 详细统计：提供详尽的统计信息，可以识别每一个数据包丢失情况。
- 支持多种场景：可用于 Layer 4 负载均衡器及其他 Layer 4 网关的负载测试与稳定性测试；云服务器网络性能测试；NIC 和 CPU 的网络包处理能力性能测试。
