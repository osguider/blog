## [wshobson/agents](https://github.com/wshobson/agents)

agents 是一个面向 Claude Code 的智能自动化与多代理编排系统，集成了丰富的专业代理、插件和开发工具，实现高效且模块化的软件开发支持。

- 提供 63 个专注单一功能的插件，优化令牌使用并支持灵活组合
- 包含 85 个领域专家级别的专业 AI 代理，覆盖架构、语言、基础设施、安全等多个方向
- 具备 47 项渐进式加载的技能包，实现知识按需调用以节省资源
- 配备 15 套多代理工作流编排器，用于复杂任务如全栈开发、安全加固及机器学习流水线管理
- 集成 44 种实用开发工具，包括项目脚手架、安全扫描和测试自动化

{{< shields path="github/stars/google/osv-scanner" alt="Github Repo Stars" >}} {{< shields path="github/license/google/osv-scanner" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/google/osv-scanner" alt="Language: `Unknown`" >}}

osv-scanner 是一个用 Go 编写的漏洞扫描工具，利用 <https://osv.dev> 提供的数据来查找影响项目依赖项的现有漏洞。

- 连接项目依赖列表与相关漏洞，提供官方支持的前端界面
- 基于开放和权威来源的信息，提高数据库质量
- 支持用户建议改进，提高 advisories 的准确性
- 使用机器可读格式存储受影响版本信息，与开发者包列表精确映射
- 减少无效通知，加快解决问题的时间

## [open-mmlab/mmagic](https://github.com/open-mmlab/mmagic)

{{< shields path="github/stars/open-mmlab/mmagic" alt="Github Repo Stars" >}} {{< shields path="github/license/open-mmlab/mmagic" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/open-mmlab/mmagic" alt="Language: `Unknown`" >}}

![demo-picture-of-mmagic](https://static.osguider.com/subject/github/open-mmlab/mmagic/4efd1837a9defcb9780041af8d5fb07e.png)

MMagic 是一个先进的多模态生成和智能创作工具箱，旨在支持图像和视频的生成与编辑。

- 提供易于使用的 API，支持文本到图像生成、图像/视频修复与增强等功能。
- 包含强大的模型库，支持 11 种新模型及 4 个新任务。
- 引入“魔法扩散”模型，实现基于 Stable Diffusion 和 Disco Diffusion 的图像生成。
- 升级框架以提高训练和推理效率，并统一各种任务的数据格式。
- 支持对多个数据集进行评估，并提供可视化工具。
