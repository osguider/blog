## [autonomousvision/mip-splatting](https://github.com/autonomousvision/mip-splatting)

{{< shields path="github/stars/autonomousvision/mip-splatting" alt="Github Repo Stars" >}} {{< shields path="github/license/autonomousvision/mip-splatting" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/autonomousvision/mip-splatting" alt="Language: `Unknown`" >}}

![demo-picture-of-mip-splatting](https://static.osguider.com/subject/github/autonomousvision/mip-splatting/051a6a138b6ccc38528585d7e197fc99.gif)

Mip-Splatting 是一种无别名的 3D 高斯点云渲染技术，旨在消除多种伪影并实现清晰的渲染效果。

- 提供 3D 平滑滤波器和 2D Mip 滤波器，提升渲染质量
- 消除了传统方法中的多个伪影
- 集成了改进的密度化指标，提高新视角合成结果
- 支持单尺度和多尺度训练与测试，适用于不同数据集
- 包含在线查看工具，可视化训练后的模型
  
## [Eugeny/tabby](https://github.com/Eugeny/tabby)

{{< shields path="github/stars/Eugeny/tabby" alt="Github Repo Stars" >}} {{< shields path="github/license/Eugeny/tabby" alt="License: `MIT`" >}} {{< shields path="github/languages/top/Eugeny/tabby" alt="Language: `Unknown`" >}}

![demo-picture-of-tabby](https://picgo-daily.oss-cn-guangzhou.aliyuncs.com/picgo-daily/2025/991b7ebdc8e3db7572e9e83fc56576d1.png)

Tabby 是一个现代化的终端模拟器，支持 SSH 和串口连接，适用于 Windows 10、macOS 和 Linux。

- 高度可配置的终端模拟器
- 集成 SSH 和 Telnet 客户端及连接管理器
- 支持多种主题和颜色方案
- 完全可配置的快捷键和多重组合快捷键
- 多窗格分屏功能，记忆标签页状态
- 支持 PowerShell、WSL、Git-Bash 等环境
- 直接通过 Zmodem 实现文件传输
- 完整 Unicode 支持，包括双宽字符
  
## [samuelmtimbo/unit](https://github.com/samuelmtimbo/unit)

{{< shields path="github/stars/samuelmtimbo/unit" alt="Github Repo Stars" >}} {{< shields path="github/license/samuelmtimbo/unit" alt="License: `MIT`" >}} {{< shields path="github/languages/top/samuelmtimbo/unit" alt="Language: `Unknown`" >}}

![demo-picture-of-unit](https://static.osguider.com/subject/github/samuelmtimbo/unit/c3abe5557a958c6fe5172ddf4525f8f9.gif)

Unit 是一个下一代可视化编程系统，旨在提升开发者体验。

- 采用多输入多输出的有限状态机（MIMO FSM）模型，以图形方式表示程序。
- 支持通过绘图、手势和语音进行部分编辑，提供直观的操作感受。
- 类似于 Unix 命令行界面，将单位（命令）的输入和输出连接成图形脚本。
- 设计为简单易用的 Web 操作系统，促进软件共享与协作。
- 可作为通用库在任何 JavaScript 项目中使用，包括 Web 和 Node.js 环境。
