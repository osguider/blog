Thorium-Win 是一个基于 Chromium 的 Windows 版本，命名源自放射性元素 No.9

- 提供最新稳定版的 Chromium 构建
- 类似 Google Chrome 的功能和用户体验

- 提供可移植的 .zip 文件和 .exe 安装程序

## [WisdomShell/codeshell](https://github.com/WisdomShell/codeshell)

{{< shields path="github/stars/WisdomShell/codeshell" alt="Github Repo Stars" >}} {{< shields path="github/license/WisdomShell/codeshell" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/WisdomShell/codeshell" alt="Language: `Unknown`" >}}

![demo-picture-of-codeshell](https://static.osguider.com/subject/github/WisdomShell/codeshell/f86cb3d87904e608b1edec7fe6605c75.png)

CodeShell 是由北京大学知识计算实验室与四川天府银行 AI 团队联合研发的多语言代码大模型基座。

- 强大的性能，在 HumanEval 和 MBPP 上取得同类模型最佳成绩
- 提供完整的开源技术体系，包括 IDE 插件（VS Code 与 JetBrains）
- 支持轻量化本地 C++ 部署，适合个人电脑使用
- 多任务评测体系覆盖常见软件开发活动，如代码生成、缺陷检测等
- 高效训练，仅用五千亿 Token 便获得优异表现

## [DeviceFarmer/stf](https://github.com/DeviceFarmer/stf)

{{< shields path="github/stars/DeviceFarmer/stf" alt="Github Repo Stars" >}} {{< shields path="github/license/DeviceFarmer/stf" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/DeviceFarmer/stf" alt="Language: `Unknown`" >}}

![demo-picture-of-stf](https://static.osguider.com/subject/github/DeviceFarmer/stf/2fe732c2da4c6156d050626d5dd17b88.jpg)

stf 是一个用于通过浏览器远程控制和管理 Android 设备的 Web 应用程序。

- 支持多种 Android 版本（2.3.3 至 12）及其他基于 Android 的系统
- 实时屏幕查看，刷新速度可达 30-40 FPS
- 允许从本地键盘输入文本，并支持复制粘贴功能
- 多点触控支持和拖放安装 APK 文件功能
- 提供 adb 连接、执行 shell 命令以及实时输出显示
- 设备日志过滤与监控，跟踪电池状态与健康状况
- 包含预定和分区系统以便于设备管理与用户分配
  
## [langchain-ai/langserve](https://github.com/langchain-ai/langserve)

{{< shields path="github/stars/langchain-ai/langserve" alt="Github Repo Stars" >}} {{< shields path="github/license/langchain-ai/langserve" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/langchain-ai/langserve" alt="Language: `Unknown`" >}}

![demo-picture-of-langserve](https://osguider.oss-cn-guangzhou.aliyuncs.com/subject/ae653deb400ed57161fec334bd29bdd1.png)

LangServe 是一个帮助开发者将 LangChain 可运行对象和链部署为 REST API 的工具。

- 自动推断输入输出模式，并在每次 API 调用时强制执行，提供丰富的错误信息
- 提供高效的 /invoke、/batch 和 /stream 端点，支持单个服务器上的多个并发请求
- 支持流式日志记录和事件流，无需解析输出
- 内置可选追踪功能，可以与 LangSmith 集成
- 使用经过验证的开源 Python 库构建，如 FastAPI 和 Pydantic
