## [p1ngul1n0/blackbird](https://github.com/p1ngul1n0/blackbird)

{{< shields path="github/stars/p1ngul1n0/blackbird" alt="Github Repo Stars" >}} {{< shields path="github/license/p1ngul1n0/blackbird" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/p1ngul1n0/blackbird" alt="Language: `Unknown`" >}}

![demo-picture-of-blackbird](https://static.osguider.com/subject/github/p1ngul1n0/blackbird/1a86d3be65405eddb4dbcbc100435b34.png)

blackbird 是一个开源的 OSINT 工具，主要用于通过用户名和邮箱在社交网络及其他平台上搜索账户信息。

- 支持在 600 多个平台上快速进行用户名和邮箱搜索
- 集成了社区驱动项目 WhatsMyName，保证低误报率和高质量结果
- 提供智能过滤器以及 PDF/CSV 格式的导出功能
- 内置免费 AI 引擎，可对用户行为与技术特征进行分析并生成用户画像
- 完全自动化操作，通过命令行界面使用方便快捷
- AI 分析结果可包含在导出的报告中，提高信息洞察力
  
## [kyverno/kyverno](https://github.com/kyverno/kyverno)

{{< shields path="github/stars/kyverno/kyverno" alt="Github Repo Stars" >}} {{< shields path="github/license/kyverno/kyverno" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/kyverno/kyverno" alt="Language: `Unknown`" >}}

{{< github-opengraph user="kyverno" repo="kyverno" alt="cover" >}}

kyverno 是一个面向 Kubernetes 平台工程团队的原生策略引擎，旨在通过代码化策略实现安全、合规、自动化和治理管理。

- 支持使用 Kubernetes 准入控制和后台扫描对资源进行验证、变更、生成及清理
- 能够验证容器镜像签名以保障供应链安全
- 与常用工具如 kubectl、kustomize 和 Git 无缝集成
- 提供丰富的生产就绪策略库，涵盖安全合规（如 Pod 安全标准）、运维优化（自动标签命名规范）、成本控制（资源配额限制）及开发者保护措施等多种场景
- 包含详细文档、多媒体教程以及社区支持渠道，方便用户快速上手与问题解决
  
## [Arindam200/awesome-ai-apps](https://github.com/Arindam200/awesome-ai-apps)

{{< shields path="github/stars/Arindam200/awesome-ai-apps" alt="Github Repo Stars" >}} {{< shields path="github/license/Arindam200/awesome-ai-apps" alt="License: `MIT`" >}} {{< shields path="github/languages/top/Arindam200/awesome-ai-apps" alt="Language: `Unknown`" >}}

{{< github-opengraph user="Arindam200" repo="awesome-ai-apps" alt="cover" >}}

awesome-ai-apps 是一个展示基于大型语言模型（LLM）构建的多样化人工智能应用项目集合。

- 包含丰富的实用示例、教程和开发指南，涵盖从简单聊天机器人到复杂 AI 代理的多种应用场景。
- 涵盖 RAG（检索增强生成）、多智能体协作、工作流自动化等前沿技术与用例。
- 提供多个快速入门和实际案例，如金融数据跟踪、人机交互安全代理、日程管理及网页自动化等。
- 支持 Model Context Protocol (MCP) 的各种集成示范，包括文档问答、数据库对话及酒店预订等功能。
- 集成先进记忆机制，提升代理持续性和个性化表现，如研究助手与社交媒体自动化工具。
- 配套 AWS Strands 课程，系统讲解从基础到生产环境下 AI 代理开发的方法论。
