## [devlikeapro/waha](https://github.com/devlikeapro/waha)

{{< shields path="github/stars/devlikeapro/waha" alt="Github Repo Stars" >}} {{< shields path="github/license/devlikeapro/waha" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/devlikeapro/waha" alt="Language: `Unknown`" >}}

{{< github-opengraph user="devlikeapro" repo="waha" alt="cover" >}}

waha 是一个基于 HTTP REST API 的 WhatsApp 接口，允许用户在自己的服务器上快速部署并通过 API 发送 WhatsApp 消息。

- 支持三种引擎：WEBJS（浏览器端）、NOWEB（Node.js websocket）和 GOWS（Go websocket），提供多样化的运行环境选择
- 通过 Docker 容器一键安装，简化部署流程，5 分钟内即可启动服务
- 提供完整的 Swagger 文档和示例仪表盘，方便开发者测试和集成 API
- 支持多会话管理，可同时操作多个 WhatsApp 账号
- 具备扫码登录功能，通过手机扫描二维码完成身份验证后即可发送消息
- 提供文本消息发送接口，并支持 curl 命令行调用，实现灵活自动化操作

- 支持自动收集大量多源、多模态数据（包括截图、文本等），并进行智能存储管理，减轻用户负担
- 主动推送关键信息，如每日/每周总结、待办事项及活动记录，提高信息利用效率
- 智能呈现相关内容，在创作过程中辅助而不干扰用户思路
- 注重隐私保护，所有数据均本地存储确保安全性
- 采用完整的上下文工程架构，实现从捕获到消费的数据全生命周期管理

## [ShinoKana/multipleWindow3dScene](https://github.com/ShinoKana/multipleWindow3dScene)

{{< shields path="github/stars/ShinoKana/multipleWindow3dScene" alt="Github Repo Stars" >}} {{< shields path="github/license/ShinoKana/multipleWindow3dScene" alt="License: `MIT`" >}} {{< shields path="github/languages/top/ShinoKana/multipleWindow3dScene" alt="Language: `Unknown`" >}}

![demo-picture-of-multipleWindow3dScene](https://static.osguider.com/subject/github/ShinoKana/multipleWindow3dScene/1dcc4375e7b52f9be76c67f8e9185187.png)

multipleWindow3dScene 是一个基于 three.js 和 localStorage 的多窗口 3D 场景示例。

- 多窗口同步：使用 localStorage 实现多个浏览器窗口之间的 3D 场景状态同步。
- three.js 集成：利用 three.js 强大的渲染能力展示 3D 图形。
- 响应式设计：场景能够根据不同的窗口大小进行调整，保持视觉一致性。
