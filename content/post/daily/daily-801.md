## [hmjz100/LinkSwift](https://github.com/hmjz100/LinkSwift)

- 提供美化的用户界面和丰富的下载管理功能，如 AB Download Manager 集成及 Aria2 推送支持
- 持续更新以适配各个平台的新版本和修复相关 Bug，提高稳定性与兼容性
- 支持多服务配置切换，优化脚本执行效率并增强用户体验
  
## [charmbracelet/glow](https://github.com/charmbracelet/glow)

{{< shields path="github/stars/charmbracelet/glow" alt="Github Repo Stars" >}} {{< shields path="github/license/charmbracelet/glow" alt="License: `MIT`" >}} {{< shields path="github/languages/top/charmbracelet/glow" alt="Language: `Unknown`" >}}

{{< github-opengraph user="charmbracelet" repo="glow" alt="cover" >}}

glow 是一个基于终端的 Markdown 阅读器，旨在提升命令行界面中 Markdown 文件的展示效果。

- 支持在命令行直接渲染和浏览本地或 Git 仓库中的 Markdown 文件
- 提供交互式文本用户界面（TUI），方便文件查找和阅读
- 支持从文件、标准输入、GitHub/GitLab 仓库及 HTTP 链接加载 Markdown 内容
- 可自定义输出样式，包括自动适配终端背景色的暗黑/明亮主题以及自定义 JSON 样式表
- 具备分页功能，支持与系统默认或指定分页程序结合使用以优化阅读体验
- 多平台支持，提供多种包管理器安装方式及预编译二进制版本覆盖主流操作系统
  
## [mudler/edgevpn](https://github.com/mudler/edgevpn)

{{< shields path="github/stars/mudler/edgevpn" alt="Github Repo Stars" >}} {{< shields path="github/license/mudler/edgevpn" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/mudler/edgevpn" alt="Language: `Unknown`" >}}

{{< github-opengraph user="mudler" repo="edgevpn" alt="cover" >}}

edgevpn 是一个基于 libp2p 构建的去中心化、不可变且静态编译的点对点 VPN 和反向代理工具。

- 完全去中心化，无需中央服务器，支持通过共享令牌自动发现节点
- 支持创建安全的 P2P VPN 网络，自动分配 IP 地址，并内置轻量级 DNS 服务器解析内部/外部 IP
- 可建立可信区域以防止令牌泄露导致网络访问风险
- 提供反向代理功能，可将 TCP 服务暴露给 P2P 网络中的节点，无需建立完整 VPN 连接
- 支持在节点间通过 P2P 直接安全传输文件，不依赖传统 VPN 连接
- 作为 Go 语言库可嵌入其他项目，实现分布式账本等功能，如 LocalAI 的点对点特性即由其驱动
- 提供桌面 GUI（Linux alpha 版）及丰富仪表盘界面方便管理和监控网络状态

adk-docs 是一个面向开发者的开源工具包，旨在以代码优先的方式灵活构建、评估和部署复杂的 AI 代理系统。

- 提供丰富的工具生态，包括预置工具、自定义函数及 OpenAPI 规范，支持与 Google 生态紧密集成
- 支持 Python 和 Java 的代码优先开发模式，实现代理逻辑、工具和编排的一体化定义，便于测试与版本管理
- 支持模块化多代理系统设计，可通过组合多个专业化代理构建可扩展且灵活的层级结构
- 内置追踪与监控功能，实现对代理行为和工作流调试优化的可观察性
- 可容器化部署于 Cloud Run、GKE 或借助 Vertex AI Agent Engine 实现无缝弹性扩展
