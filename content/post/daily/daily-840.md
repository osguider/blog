## [epfLLM/meditron](https://github.com/epfLLM/meditron)

{{< shields path="github/stars/epfLLM/meditron" alt="Github Repo Stars" >}} {{< shields path="github/license/epfLLM/meditron" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/epfLLM/meditron" alt="Language: `Unknown`" >}}

![demo-picture-of-meditron](https://static.osguider.com/subject/github/epfLLM/meditron/1f46d602ce375f5bbe18ebfcb3714ba3.png)

Meditron 是一套开源的医疗大型语言模型（LLMs）。

- 提供 Meditron-7B 和 Meditron-70B 两个版本，专为医疗领域适配。
- 通过对精选医学语料库进行持续预训练，提升了模型在医学推理任务上的表现。
- 在多个医学推理任务中超越 Llama-2、GPT-3.5 和 Flan-PaLM 的性能。
- 包含来自临床指南、论文摘要和公开可用文献的数据集，用于训练和微调。
  
## [Ma-Lab-Berkeley/CRATE](https://github.com/Ma-Lab-Berkeley/CRATE)

{{< shields path="github/stars/Ma-Lab-Berkeley/CRATE" alt="Github Repo Stars" >}} {{< shields path="github/license/Ma-Lab-Berkeley/CRATE" alt="License: `MIT`" >}} {{< shields path="github/languages/top/Ma-Lab-Berkeley/CRATE" alt="Language: `Unknown`" >}}

![demo-picture-of-CRATE](https://static.osguider.com/subject/github/Ma-Lab-Berkeley/CRATE/d6fc287658b8591acae0cdbff79c823c.png)

CRATE 是一个白盒变换器架构，旨在通过稀疏率减少目标进行优化。

- 采用 PyTorch 实现，支持多种实验和模型训练。
- 每层执行交替最小化算法，以优化稀疏率减少目标。
- 提供预训练检查点，可用于 ImageNet 和 CIFAR10 数据集的微调与训练。
- 包含自动编码功能，并展示了 CRATE 在分割任务中的表现。
