## [rails/solid_queue](https://github.com/rails/solid_queue)

{{< shields path="github/stars/rails/solid_queue" alt="Github Repo Stars" >}} {{< shields path="github/license/rails/solid_queue" alt="License: `MIT`" >}} {{< shields path="github/languages/top/rails/solid_queue" alt="Language: `Unknown`" >}}

{{< github-opengraph user="rails" repo="solid_queue" alt="cover" >}}

solid_queue 是一个基于数据库的 Active Job 后端，旨在提供简单性和性能。

- 支持常规作业排队和处理
- 支持延迟作业、并发控制、定期作业以及暂停队列
- 提供每个作业的数字优先级及按队列顺序的优先级
- 批量排队功能（enqueue_all）
- 兼容 MySQL、PostgreSQL 和 SQLite 等 SQL 数据库
- 利用 FOR UPDATE SKIP LOCKED 子句避免锁阻塞问题
- 与 Ruby on Rails 的多线程兼容，支持重试、丢弃和错误处理等功能
  
## [JShollaj/awesome-llm-interpretability](https://github.com/JShollaj/awesome-llm-interpretability)

{{< shields path="github/stars/JShollaj/awesome-llm-interpretability" alt="Github Repo Stars" >}} {{< shields path="github/license/JShollaj/awesome-llm-interpretability" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/JShollaj/awesome-llm-interpretability" alt="Language: `Unknown`" >}}

{{< github-opengraph user="JShollaj" repo="awesome-llm-interpretability" alt="cover" >}}

awesome-llm-interpretability 是一个整理的大型语言模型（LLM）可解释性资源列表。
该项目的主要功能、关键特性、核心优势：

- 提供了各种工具、论文、文章和社区，专注于大型语言模型（LLM）可解释性。
- 包含了多种工具和库，用于 LLM 的可解释性分析。
- 支持分类、回归和生成模型（文本与图像数据），包括显著方法、注意力归因法等。
- 提供了各种学术及行业论文，探讨 LLM 可解释性问题。

## [agalwood/Motrix](https://github.com/agalwood/Motrix)

{{< shields path="github/stars/agalwood/Motrix" alt="Github Repo Stars" >}} {{< shields path="github/license/agalwood/Motrix" alt="License: `MIT`" >}} {{< shields path="github/languages/top/agalwood/Motrix" alt="Language: `Unknown`" >}}

![demo-picture-of-Motrix](https://static.osguider.com/subject/github/agalwood/Motrix/778c2f3c56519e07559d8f6f82913d58.png)

Motrix 是一款功能齐全的下载管理器。

- 简洁明了的用户界面
- 支持 HTTP、FTP、BitTorrent 和 Magnet 下载
- BitTorrent 选择性下载功能
- 每日自动更新追踪器列表
- UPnP 和 NAT-PMP 端口映射支持
- 最多可同时进行 10 个下载任务
- 单个任务支持 64 线程
- 限速功能
- 模拟用户代理
- 下载完成通知
  
## [docker-easyconnect/docker-easyconnect](https://github.com/docker-easyconnect/docker-easyconnect)

{{< shields path="github/stars/docker-easyconnect/docker-easyconnect" alt="Github Repo Stars" >}} {{< shields path="github/license/docker-easyconnect/docker-easyconnect" alt="License: `WTFPL`" >}} {{< shields path="github/languages/top/docker-easyconnect/docker-easyconnect" alt="Language: `Unknown`" >}}

{{< github-opengraph user="docker-easyconnect" repo="docker-easyconnect" alt="cover" >}}

docker-easyconnect 是一个使深信服（Sangfor）开发的非自由 VPN 软件 EasyConnect 和 aTrust 运行在 Docker 或 Podman 中的项目，提供网关和代理服务。

- 支持通过 Docker 部署 EasyConnect 和 aTrust
- 提供 SOCKS5 和 HTTP 代理服务
- 可作为网络网关供宿主机连接使用
- 支持多种架构，包括 amd64、arm64 等
- 简化了 VPN 登录过程，支持命令行和图形界面两种方式
