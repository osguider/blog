## [enactic/openarm](https://github.com/enactic/openarm)

{{< shields path="github/stars/enactic/openarm" alt="Github Repo Stars" >}} {{< shields path="github/license/enactic/openarm" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/enactic/openarm" alt="Language: `Unknown`" >}}

![demo-picture-of-openarm](https://github.com/enactic/openarm/raw/main/website/static/img/teleop/leader-follower/unilateral.png)

openarm 是一个完全开源的 7 自由度类人机械臂，专为物理人工智能研究和在复杂接触环境中的应用而设计。

- 具有人体比例尺寸，适合模拟和实际操作
- 高背驱动性与顺应性，确保安全的人机交互
- 支持实用负载能力，可满足真实世界任务需求
- 提供完整的硬件 CAD 数据及软件代码，包括 ROS2 集成、CAN 通信库等组件
- 适用于远程操控、模仿学习、仿真及数据采集等多种场景
- 成本相对低廉（双臂系统约 6500 美元），便于科研与开发使用
  
## [guofei9987/scikit-opt](https://github.com/guofei9987/scikit-opt)

{{< shields path="github/stars/guofei9987/scikit-opt" alt="Github Repo Stars" >}} {{< shields path="github/license/guofei9987/scikit-opt" alt="License: `MIT`" >}} {{< shields path="github/languages/top/guofei9987/scikit-opt" alt="Language: `Unknown`" >}}

{{< github-opengraph user="guofei9987" repo="scikit-opt" alt="cover" >}}

scikit-opt 是一个基于 Python 的群智能优化算法库，集成了遗传算法、粒子群优化、模拟退火、蚁群算法、免疫算法和人工鱼群算法等多种优化方法，并支持解决旅行商问题（TSP）。

- 支持用户自定义函数（UDF），允许用户灵活定制选择算子、交叉算子和变异算子等操作。
- 提供丰富的内置操作符，方便快速构建和调整遗传算法流程。
- 支持断点续跑功能，可分阶段运行并继续迭代，提高实验灵活性。
- 具备多种加速方式，包括向量化、多线程、多进程及缓存机制，以提升计算效率。
- 正在开发 GPU 计算支持，进一步增强大规模复杂问题求解能力。
