## [GongRzhe/Office-Word-MCP-Server](https://github.com/GongRzhe/Office-Word-MCP-Server)

{{< shields path="github/stars/GongRzhe/Office-Word-MCP-Server" alt="Github Repo Stars" >}} {{< shields path="github/license/GongRzhe/Office-Word-MCP-Server" alt="License: `MIT`" >}} {{< shields path="github/languages/top/GongRzhe/Office-Word-MCP-Server" alt="Language: `Unknown`" >}}

{{< github-opengraph user="GongRzhe" repo="Office-Word-MCP-Server" alt="cover" >}}

Office-Word-MCP-Server 是一个基于模型上下文协议（MCP），用于创建、读取和操作微软 Word 文档的服务器，旨在通过标准化接口支持 AI 助手对 Word 文档的丰富编辑功能。

- 提供文档管理功能，包括新建带元数据的 Word 文件、提取文本与结构分析、查看属性统计、目录中文件列表管理及多文档合并与格式转换
- 支持内容创建，涵盖多级标题添加、多样段落插入、自定义表格构建、图片比例缩放以及分页符和注释（脚注/尾注）灵活处理
- 丰富文本格式控制，实现文字加粗斜体下划线、多重字体颜色及样式应用，并支持全文搜索替换和单元格内独立格式设置
- 细致表格设计能力，包括边框样式调整、行列合并、多色交替背景、高级单元格填充与对齐，以及自动适应宽度等专业排版特性
- 高级文档操作，如删除段落以及基于文本或段落索引精确插入内容，同时保证编号列表符合 XML 标准规范

snapDOM 是一个高效且精准的 HTML 元素截图工具，能够将任意 DOM 节点捕获为可缩放的 SVG 图像，同时支持导出为多种光栅格式和 Canvas。

- 支持完整 DOM 捕获，包括样式、字体、背景图像、伪元素及 Shadow DOM
- 导出格式丰富，涵盖 SVG、PNG、JPG、WebP 以及 Canvas 和 Blob
- 速度极快，无任何依赖，基于标准 Web API 实现
- 提供灵活的 API 接口与快捷方法，方便开发者调用和集成
- 支持自定义输出尺寸、多种压缩选项及字体内嵌功能

## [quarto-dev/quarto-cli](https://github.com/quarto-dev/quarto-cli)

{{< shields path="github/stars/quarto-dev/quarto-cli" alt="Github Repo Stars" >}} {{< shields path="github/license/quarto-dev/quarto-cli" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/quarto-dev/quarto-cli" alt="Language: `Unknown`" >}}

{{< github-opengraph user="quarto-dev" repo="quarto-cli" alt="cover" >}}

quarto-cli 是一个基于 Pandoc 构建的开源科学与技术出版系统。

- 支持使用 Markdown 编写文档，简化文本创作流程
- 集成 Jupyter、Knitr 和 Observable，实现 Python、R、Julia 及 JavaScript 代码和输出的嵌入
- 扩展 Pandoc Markdown，提供交叉引用、副图、多栏布局、悬浮引用脚注和提示框等技术写作功能
- 提供项目管理系统，可批量渲染文档并生成网站或书籍等聚合输出
- 支持多种编辑环境，包括 JupyterLab、RStudio 和 VS Code，以及内置可视化 Markdown 编辑器
