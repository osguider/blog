## [pd4d10/console-importer](https://github.com/pd4d10/console-importer)

{{< shields path="github/stars/pd4d10/console-importer" alt="Github Repo Stars" >}} {{< shields path="github/license/pd4d10/console-importer" alt="License: `MIT`" >}} {{< shields path="github/languages/top/pd4d10/console-importer" alt="Language: `Unknown`" >}}

![demo-picture-of-console-importer](https://static.osguider.com/subject/github/pd4d10/console-importer/5de9102f2af7e527e6cb7356d67a2f90.gif)

console-importer 是一个可以轻松从 Chrome 控制台导入 JS 和 CSS 资源的工具。

- 支持通过控制台命令 `$i` 导入 JavaScript 和 CSS 资源
- 可以指定特定版本进行导入，例如 `$i('jquery@2')`
- 支持直接使用有效的脚本 URL 导入
- 提供 ES 模块支持，避免全局变量污染
- 针对某些网站提供了替代方案以解决兼容性问题
  
## [teslamotors/fleet-telemetry](https://github.com/teslamotors/fleet-telemetry)

{{< shields path="github/stars/teslamotors/fleet-telemetry" alt="Github Repo Stars" >}} {{< shields path="github/license/teslamotors/fleet-telemetry" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/teslamotors/fleet-telemetry" alt="Language: `Unknown`" >}}

![demo-picture-of-fleet-telemetry](https://static.osguider.com/subject/github/teslamotors/fleet-telemetry/f0b5a835f1da1c209b8caefdb335567f.png)

fleet-telemetry 是一个用于特斯拉遥测协议的服务器参考实现，旨在定期获取特斯拉车辆的数据。车主可以允许注册的应用程序安全地直接从他们的车辆接收遥测数据。

- 提供设备连接管理和数据接收、存储功能
- 支持通过 WebSocket 推送可配置的遥测记录
- 允许个体车主和车队运营商灵活控制数据传输时间和条件
- 提供事件流以确定车辆连接状态，作为在线状态代理
- 推荐在 Kubernetes 环境中安装，以简化操作与维护
