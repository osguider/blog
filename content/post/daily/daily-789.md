## [DigitalPlatDev/FreeDomain](https://github.com/DigitalPlatDev/FreeDomain)

{{< shields path="github/stars/DigitalPlatDev/FreeDomain" alt="Github Repo Stars" >}} {{< shields path="github/license/DigitalPlatDev/FreeDomain" alt="License: `AGPL-3.0`" >}} {{< shields path="github/languages/top/DigitalPlatDev/FreeDomain" alt="Language: `Unknown`" >}}

{{< github-opengraph user="DigitalPlatDev" repo="FreeDomain" alt="cover" >}}

FreeDomain 是一个提供免费域名注册服务的平台，旨在让每个人都能拥有数字身份，无需支付任何费用。

- 提供多种免费顶级域名后缀，如 .DPDNS.ORG、.US.KG、.QZZ.IO 和 .XX.KG，且未来将增加更多选项
- 用户可自由选择喜欢的 DNS 服务商进行托管，包括 Cloudflare、FreeDNS 等
- 已有超过 35 万个域名注册用户，覆盖个人和组织群体，具有较高的信任度和社区支持
- 由非营利机构 The Hack Foundation 财政赞助，以确保项目持续运营并保持开放性与公益性
- 提供详细教程及活跃社区支持，通过 Telegram、Discord 和 GitHub 讨论区方便用户交流与获取帮助
- 重视滥用举报管理，致力于维护安全开放的互联网环境

- 支持多语言（Java、Python、Javascript）的自动化测试用例生成
- 自动补全缺失的 Assert，提高现有测试用例的有效性
- 提供本地化部署选项，确保数据安全无泄露风险
- 采用业界领先的 TestGPT-7B 模型，具备高通过率和场景覆盖能力

![demo-picture-of-RustyTube](https://static.osguider.com/subject/github/opensourcecheemsburgers/RustyTube/0e04e513d01a00511ff51b6111a49af3.png)

RustyTube 是一个用 Rust 编写的桌面 YouTube 客户端。

- 支持 4K 60fps 的 WebM 视频格式
- 显示不喜欢计数
- 拥有现代美观的用户界面
- 提供超过 30 种主题，包括基于 Sweet Ambar Blue Dark 的自定义主题
- 不使用 Electron 框架，性能更优
  
## [Chuny1/3DGPT](https://github.com/Chuny1/3DGPT)

{{< shields path="github/stars/Chuny1/3DGPT" alt="Github Repo Stars" >}} {{< shields path="github/license/Chuny1/3DGPT" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/Chuny1/3DGPT" alt="Language: `Unknown`" >}}

![demo-picture-of-3DGPT](https://static.osguider.com/subject/github/Chuny1/3DGPT/b63d0e9582e2c94631150ba169095d14.png)

3DGPT 是一个利用大型语言模型（LLM）实现程序化 3D 建模的开源项目。

- 提供了核心智能代理 (agent) 逻辑，能够生成 Python 脚本以控制 Infinigen 进行 3D 场景生成
- 支持多种代理之间的交互及函数参数推断，增强了对复杂任务的处理能力
- 计划发布改进版 Infinigen 和更友好的人机协作配置文件解析器，提高适配性和易用性
- 基于 OpenAI API，可灵活选择不同 GPT 模型进行调用，实现自然语言驱动的三维建模流程
- 项目附带详细示例文档及使用说明，有助于理解与快速上手当前功能
