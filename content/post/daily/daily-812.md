## [kijai/ComfyUI-WanVideoWrapper](https://github.com/kijai/ComfyUI-WanVideoWrapper)

{{< shields path="github/stars/kijai/ComfyUI-WanVideoWrapper" alt="Github Repo Stars" >}} {{< shields path="github/license/kijai/ComfyUI-WanVideoWrapper" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/kijai/ComfyUI-WanVideoWrapper" alt="Language: `Unknown`" >}}

![demo-picture-of-ComfyUI-WanVideoWrapper](https://static.osguider.com/subject/github/kijai/ComfyUI-WanVideoWrapper/7d4109f74a6a25f3d74351a909304b72.png)

ComfyUI-WanVideoWrapper 是一个用于集成 WanVideo 及相关模型的 ComfyUI 自定义节点包装器，旨在作为个人测试和快速实现新模型与功能的沙盒环境。

- 提供对 WanVideo 及多种视频生成和处理模型（如 ReCamMaster、VACE、Phantom 等）的支持
- 允许使用 FP8 缩放模型以优化性能
- 支持将文本编码器、Clip 视觉模块等加载到 ComfyUI 中，兼容原生组件或替代方案
- 方便用户快速尝试新发布的功能，无需担心核心代码兼容性问题
- 包含示例演示不同应用场景下的视频生成效果与参数调整方法
- 持续开发中，适合探索最新技术但可能存在不稳定因素
  
## [steveiliop56/tinyauth](https://github.com/steveiliop56/tinyauth)

{{< shields path="github/stars/steveiliop56/tinyauth" alt="Github Repo Stars" >}} {{< shields path="github/license/steveiliop56/tinyauth" alt="License: `GPL-3.0`" >}} {{< shields path="github/languages/top/steveiliop56/tinyauth" alt="Language: `Unknown`" >}}

{{< github-opengraph user="steveiliop56" repo="tinyauth" alt="cover" >}}

tinyauth 是一个用于为应用程序添加登录保护的简单认证中间件。

- 支持通过简单的登录界面或集成 Google、Github 等 OAuth 提供商进行身份验证
- 兼容主流代理服务器如 Traefik、Nginx 和 Caddy，方便在 Docker 环境中部署
- 提供易于上手的文档和示例 docker-compose 配置，便于快速体验与使用
- 项目处于活跃开发阶段，持续更新功能与配置选项
- 包含在线演示版本，可直接测试默认用户名密码以评估适用性
  
## [idosal/git-mcp](https://github.com/idosal/git-mcp)

{{< shields path="github/stars/idosal/git-mcp" alt="Github Repo Stars" >}} {{< shields path="github/license/idosal/git-mcp" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/idosal/git-mcp" alt="Language: `Unknown`" >}}

{{< github-opengraph user="idosal" repo="git-mcp" alt="cover" >}}

git-mcp 是一个免费开源的远程模型上下文协议（MCP）服务器，旨在将任何 GitHub 项目转变为文档中心，从而帮助 AI 工具访问最新的代码和文档，减少代码幻觉现象。

- 支持特定仓库模式和通用服务器模式，满足不同使用场景下对项目访问的灵活需求
- 允许 AI 助手直接获取最新源码及 API 文档，提高回答准确性与相关性
- 云端运行，无需本地安装或注册，即插即用，简化配置流程
- 内置智能搜索功能，高效定位所需信息，同时节省令牌消耗
- 提供内嵌聊天界面，可直接通过浏览器与仓库文档交互查询
- 完全开源且免费，不收集用户个人信息，也支持自托管部署
  
## [open-telemetry/opentelemetry-collector-contrib](https://github.com/open-telemetry/opentelemetry-collector-contrib)

{{< shields path="github/stars/open-telemetry/opentelemetry-collector-contrib" alt="Github Repo Stars" >}} {{< shields path="github/license/open-telemetry/opentelemetry-collector-contrib" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/open-telemetry/opentelemetry-collector-contrib" alt="Language: `Unknown`" >}}

{{< github-opengraph user="open-telemetry" repo="opentelemetry-collector-contrib" alt="cover" >}}

opentelemetry-collector-contrib 是 OpenTelemetry Collector 的一个扩展组件仓库，包含了不适合放入核心仓库的各种采集器组件。

- 提供多样化的采集器组件，支持追踪、指标和日志等多种信号类型
- 组件稳定性分级管理，不同信号可能处于不同开发阶段（如稳定、Alpha 或开发中）
- 部分功能通过特性门控控制，以便逐步引入主代码路径
- 由社区维护者及特定厂商共同支持，确保各个组件持续更新与安全
- 支持用户基于核心和贡献仓库中的模块构建自定义的 OpenTelemetry Collector 分发版本
  
## [awesomedata/awesome-public-datasets](https://github.com/awesomedata/awesome-public-datasets)

{{< shields path="github/stars/awesomedata/awesome-public-datasets" alt="Github Repo Stars" >}} {{< shields path="github/license/awesomedata/awesome-public-datasets" alt="License: `MIT`" >}} {{< shields path="github/languages/top/awesomedata/awesome-public-datasets" alt="Language: `Unknown`" >}}

{{< github-opengraph user="awesomedata" repo="awesome-public-datasets" alt="cover" >}}

awesome-public-datasets 是一个以主题为中心的高质量公共数据集列表项目。

- 收集并整理了来自博客、问答和用户反馈的多领域公开数据资源
- 涵盖农业、建筑等多个专业领域，提供丰富的数据样本
- 大部分数据免费开放，便于研究和开发使用
- 由上海交通大学 OMNILab 孵化，并与 BaiYuLan 开源社区合作维护
- 数据库自动生成，保持内容更新及时且规范
