## [zhayujie/chatgpt-on-wechat](https://github.com/zhayujie/chatgpt-on-wechat)

{{< shields path="github/stars/zhayujie/chatgpt-on-wechat" alt="Github Repo Stars" >}} {{< shields path="github/license/zhayujie/chatgpt-on-wechat" alt="License: `MIT`" >}} {{< shields path="github/languages/top/zhayujie/chatgpt-on-wechat" alt="Language: `Unknown`" >}}

![demo-picture-of-chatgpt-on-wechat](https://static.osguider.com/subject/github/zhayujie/chatgpt-on-wechat/c288e393002bf7875731d223a0ba986a.png)

chatgpt-on-wechat 是一个基于大模型的智能聊天机器人，支持多种平台接入。

- 多端部署：支持微信公众号、企业微信应用、飞书和钉钉等多种接入方式。
- 基础对话：提供私聊及群聊的智能回复，支持多轮会话上下文记忆。
- 语音能力：可识别并回复语音消息，兼容多个语音模型。
- 图像能力：具备图片生成与识别功能，并能进行图生图处理。
- 丰富插件：允许个性化插件扩展，实现多角色切换和联网搜索等功能。
- 知识库定制：通过上传知识库文件，自定义专属机器人，用于数字分身或智能客服。

## [dimitri/pgloader](https://github.com/dimitri/pgloader)

{{< shields path="github/stars/dimitri/pgloader" alt="Github Repo Stars" >}} {{< shields path="github/license/dimitri/pgloader" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/dimitri/pgloader" alt="Language: `Unknown`" >}}

{{< github-opengraph user="dimitri" repo="pgloader" alt="cover" >}}

pgloader 是一个用于将数据迁移到 PostgreSQL 的工具，支持通过单条命令完成数据加载。

- 采用 PostgreSQL 的 COPY 命令进行高效的数据导入。
- 支持事务处理，能够在遇到错误数据时跳过并记录拒绝的行，而不中断整个批量加载过程。
- 实现了数据格式转换，例如将 MySQL 中无效日期（如 0000-00-00）转换为 PostgreSQL 可接受的 NULL 值。
- 提供丰富的命令行选项和配置能力，包括日志管理、编码设置、预执行及后执行 SQL 脚本等功能。
- 支持从多种数据库源（如 SQLite、MySQL）直接迁移至 PostgreSQL，并能自动处理模式定义和索引等结构。
  
## [xbmc/xbmc](https://github.com/xbmc/xbmc)

{{< shields path="github/stars/xbmc/xbmc" alt="Github Repo Stars" >}} {{< shields path="github/license/xbmc/xbmc" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/xbmc/xbmc" alt="Language: `Unknown`" >}}

![demo-picture-of-xbmc](https://static.osguider.com/subject/github/xbmc/xbmc/97bc1b1a8e5a53a64492a2e494f79087.gif)

xbmc 是一款获奖的免费开源家庭影院/媒体中心软件，旨在为数字媒体提供娱乐服务。

- 支持多种操作系统，包括 Android、Linux、BSD、macOS、iOS、tvOS 和 Windows。
- 拥有美观的界面和强大的皮肤引擎，适合使用遥控器从沙发上操作。
- 能够播放几乎所有流行的音频和视频格式，并支持网络播放功能。
- 自动扫描并创建个性化库，包括封面图像、描述和粉丝艺术作品。
- 提供播放列表与幻灯片功能，以及天气预报等附加特性。
  
## [ShishirPatil/gorilla](https://github.com/ShishirPatil/gorilla)

{{< shields path="github/stars/ShishirPatil/gorilla" alt="Github Repo Stars" >}} {{< shields path="github/license/ShishirPatil/gorilla" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/ShishirPatil/gorilla" alt="Language: `Unknown`" >}}

![demo-picture-of-gorilla](https://static.osguider.com/subject/github/ShishirPatil/gorilla/1433666854ed273fb0f2f9dba2d07779.png)

gorilla 是一个旨在训练和评估大型语言模型（LLMs）调用函数（工具调用）的开源项目，重点在于通过自然语言查询准确生成语义和语法正确的 API 调用。

- 支持超过 1600 个不断增长的 API 调用，显著减少模型幻觉现象
- 提供完整的推理代码、评估代码及 APIBench 数据集——最大规模且易于训练的 API 集合
- 包含多轮、多步骤函数调用能力的新型状态化评测系统
- 引入了基于检索增强的方法以提升测试时适应性
- 发布了 GoEx 运行时，实现对 LLM 生成动作（如代码、API 请求等）的后验验证与风险控制机制
- 建立了 Berkeley Function Calling 排行榜，用以比较不同模型和工具在多场景下表现优劣
- 提供端到端微调方案、基础设施组件及 Gorilla API 商店支持生态发展
