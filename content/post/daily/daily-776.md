## [knownsec/aipyapp](https://github.com/knownsec/aipyapp)

{{< shields path="github/stars/knownsec/aipyapp" alt="Github Repo Stars" >}} {{< shields path="github/license/knownsec/aipyapp" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/knownsec/aipyapp" alt="Language: `Unknown`" >}}

{{< github-opengraph user="knownsec" repo="aipyapp" alt="cover" >}}

aipyapp 是一个集成大语言模型（LLM）的 Python 命令行解释器，实现了“Python use”理念，旨在让 LLM 能够像人类一样直接操作完整的 Python 运行环境来解决复杂问题。

- 提供两种运行模式：任务模式（适合不熟悉 Python 的用户）和 Python 模式（适合高级用户），支持自然语言描述需求并自动生成执行代码
- LLM 可以自由使用所有标准的 Python 功能，无需预定义工具接口，增强灵活性和扩展性
- 方便处理多种数据格式及常见数据操作，如清洗、转换、聚合、分析与可视化
- 支持自动安装第三方库以满足代码执行需求，提高使用便捷度
- 通过自然语言交互简化传统手动编写和复制粘贴代码流程，提高工作效率

claude-code-templates 是一个用于配置和监控 Anthropic 的 Claude Code 的命令行工具。

- 提供丰富的预设配置，包括 AI 代理、定制命令、设置项、钩子以及外部服务集成（MCPs），以提升开发工作流效率
- 支持通过交互式网页界面浏览和安装 100 多个组件与模板，涵盖安全审计员、性能优化器等多种专业 AI 角色
- 集成 GitHub、PostgreSQL、Stripe 等多种外部服务，实现项目自动化与扩展能力
- 包含实时分析工具，可监控 AI 开发会话状态及性能指标，并支持移动端对话实时查看及远程安全访问
- 提供健康检查功能，确保 Claude Code 环境运行最优，同时配备详尽文档指导用户使用各类组件和工具
