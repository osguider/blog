## [linyiLYi/voice-assistant](https://github.com/linyiLYi/voice-assistant)

{{< shields path="github/stars/linyiLYi/voice-assistant" alt="Github Repo Stars" >}} {{< shields path="github/license/linyiLYi/voice-assistant" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/linyiLYi/voice-assistant" alt="Language: `Unknown`" >}}

{{< github-opengraph user="linyiLYi" repo="voice-assistant" alt="cover" >}}

voice-assistant 是一个简单的本地语音助手演示项目，结合了语音识别和大型语言模型。

- 通过 Python 脚本实现与本地大型语言模型的对话功能
- 使用 Apple MLX 示例库进行语音识别
- 利用 01.AI 的 Yi 模型生成文本响应
- 支持多种操作系统环境配置，包括 macOS、Windows 和 Linux
- 提供简单工具查看可用麦克风列表
