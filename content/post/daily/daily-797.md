## [sharkdp/bat](https://github.com/sharkdp/bat)

{{< shields path="github/stars/sharkdp/bat" alt="Github Repo Stars" >}} {{< shields path="github/license/sharkdp/bat" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/sharkdp/bat" alt="Language: `Unknown`" >}}

{{< github-opengraph user="sharkdp" repo="bat" alt="cover" >}}

bat 是一个带有语法高亮和 Git 集成功能的 cat(1) 命令克隆工具。

- 支持多种编程语言和标记语言的语法高亮显示
- 与 Git 集成，能够显示文件相对于索引的修改情况
- 可展示不可打印字符，方便调试特殊字符问题
- 自动分页输出，大文件时自动调用分页器，也支持关闭分页以模拟 cat 行为
- 支持多个文件拼接输出，并能在非交互式终端下作为 cat 的替代品使用
- 提供丰富命令行选项，如指定语言、显示行号等自定义功能
- 能与 fzf、find/fd、ripgrep 等工具集成，用于预览搜索结果或持续监控日志文件并保持高亮效果
- 兼容 git show 和 git diff，可用于查看历史版本及代码变更部分并进行语法着色

IsaacSim 是一个基于 NVIDIA Omniverse 的开源机器人仿真平台，用于在逼真的虚拟环境中开发、测试和训练 AI 驱动的机器人。

- 支持从 URDF、MJCF 和 CAD 等常见格式导入机器人系统及环境资产
- 利用高保真 GPU 加速物理引擎实现精确动力学模拟与多传感器 RTX 渲染
- 提供包括合成数据生成、强化学习、ROS 集成及数字孪生仿真在内的端到端工作流程
- 具备丰富的机器人调优工具，支持物理准确性、计算效率和光线真实感优化
- 集成了控制器、运动生成与运动学求解器等功能，方便机器人的动作模拟与策略整合
