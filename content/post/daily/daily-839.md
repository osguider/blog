## [rust-lang/rustlings](https://github.com/rust-lang/rustlings)

{{< shields path="github/stars/rust-lang/rustlings" alt="Github Repo Stars" >}} {{< shields path="github/license/rust-lang/rustlings" alt="License: `MIT`" >}} {{< shields path="github/languages/top/rust-lang/rustlings" alt="Language: `Unknown`" >}}

{{< github-opengraph user="rust-lang" repo="rustlings" alt="cover" >}}

rustlings 是一个小型练习项目，旨在帮助用户熟悉阅读和编写 Rust 代码。

- 提供一系列小练习，帮助学习者掌握 Rust 编程语言。
- 包含与编译器消息互动的任务，以增强理解能力。
- 推荐与官方 Rust 书籍配合使用，以获得更全面的学习体验。
- 支持多种编辑器和现代终端，提高用户体验。
- 每个主题都有附加资源和 README.md 文件以指导学习。
  
## [abhiTronix/vidgear](https://github.com/abhiTronix/vidgear)

{{< shields path="github/stars/abhiTronix/vidgear" alt="Github Repo Stars" >}} {{< shields path="github/license/abhiTronix/vidgear" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/abhiTronix/vidgear" alt="Language: `Unknown`" >}}

![demo-picture-of-vidgear](https://static.osguider.com/subject/github/abhiTronix/vidgear/ac828d936b624ea10f365f55dff8cccc.png)

Vidgear 是一个高性能的跨平台视频处理 Python 框架，具有独特的创新功能。

- 提供易于使用且高度可扩展的多线程和异步 API 框架
- 支持实时读取、写入、处理和传输视频文件/帧/流
- 集成了多个先进库（如 OpenCV、FFmpeg 等），实现强大的错误处理和实时性能
- 旨在简化复杂的视频处理任务，仅需少量代码即可完成
- 包含多种专用 API（称为 Gears），适用于不同的数据源和设备
  
## [SUDO-AI-3D/zero123plus](https://github.com/SUDO-AI-3D/zero123plus)

{{< shields path="github/stars/SUDO-AI-3D/zero123plus" alt="Github Repo Stars" >}} {{< shields path="github/license/SUDO-AI-3D/zero123plus" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/SUDO-AI-3D/zero123plus" alt="Language: `Unknown`" >}}

![demo-picture-of-zero123plus](https://static.osguider.com/subject/github/SUDO-AI-3D/zero123plus/52512d2da91c1133aca1c33984c5474b.jpg)

zero123plus 是一个用于将单张图像转换为一致的多视角扩散基础模型的代码库。

- 支持从单一输入图像生成多个视角的图像。
- 处理相机内参更加精确，适应更广泛的输入场景。
- 提供法线生成器 ControlNet，用于生成视空间法线图像，提高掩膜准确性。
- 输出结果在验证集上的指标表现优异，如 IoU 达到 98.81%。
  
## [all-in-aigc/gpts-works](https://github.com/all-in-aigc/gpts-works)

{{< shields path="github/stars/all-in-aigc/gpts-works" alt="Github Repo Stars" >}} {{< shields path="github/license/all-in-aigc/gpts-works" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/all-in-aigc/gpts-works" alt="Language: `Unknown`" >}}

![demo-picture-of-gpts-works](https://osguider.oss-cn-guangzhou.aliyuncs.com/subject/4c559d9bcfbf89910df82f88aa2453b4.png)

gpts-works 是一个第三方 GPTs 商店。

- 提供网站用于展示和访问各种 GPTs。
- 包含索引系统，支持通过向量搜索功能查找 GPTs。
- 提供浏览器扩展，可以在 ChatGPT 探索页面旁边显示第三方 GPTs。
- 使用 Vercel 部署，确保高效的网页加载和数据存储。
- 支持本地开发与部署，方便用户自定义使用体验。
