## [anomalyco/opencode](https://github.com/anomalyco/opencode)

{{< shields path="github/stars/anomalyco/opencode" alt="Github Repo Stars" >}} {{< shields path="github/license/anomalyco/opencode" alt="License: `MIT`" >}} {{< shields path="github/languages/top/anomalyco/opencode" alt="Language: `Unknown`" >}}

{{< github-opengraph user="anomalyco" repo="opencode" alt="cover" >}}

opencode 是一个开源的人工智能编程代理，旨在为开发者提供灵活且多样化的代码辅助工具。

- 支持多平台安装，包括 macOS、Windows、Linux 及各种包管理器，方便用户快速部署
- 内置两种智能代理模式：全功能开发用“build”代理和只读分析用“plan”代理，以满足不同编码场景需求
- 采用客户端/服务器架构，实现本地运行与远程控制分离，可通过终端界面或移动端进行交互
- 完全开源且不依赖特定模型供应商，兼容 Claude、OpenAI、Google 及本地模型等多种后端选择
- 原生支持语言服务器协议（LSP）和终端用户界面（TUI），专注于提升命令行环境下的使用体验

- 提供丰富真实案例，包括远程服务器执行脚本、JSON 处理与 Cloudflare API 调用等应用场景
- 涵盖调试测试、自定义命令创建及日志解析等进阶技巧

## [MiroMindAI/MiroThinker](https://github.com/MiroMindAI/MiroThinker)

{{< shields path="github/stars/MiroMindAI/MiroThinker" alt="Github Repo Stars" >}} {{< shields path="github/license/MiroMindAI/MiroThinker" alt="License: `MIT`" >}} {{< shields path="github/languages/top/MiroMindAI/MiroThinker" alt="Language: `Unknown`" >}}

{{< github-opengraph user="MiroMindAI" repo="MiroThinker" alt="cover" >}}

MiroThinker 是一个开源搜索代理系列，致力于通过工具辅助推理提升复杂信息检索和研究能力。

- 提供原生支持工具辅助推理的搜索代理模型，在多项基准测试中表现领先，包括 HLE、BrowseComp 及 GAIA 等
- 包含完整的研究代理框架（MiroFlow）、大规模训练数据集（MiroVerse）以及高效稳定的训练基础设施（MiroTrain/MiroRL）
- 支持超长上下文窗口（256K），实现深度多步分析与长远推理，单任务可调用 400 次以上工具
- 采用交互式扩展策略，将模型大小、上下文长度之外引入环境交互频率作为性能提升新维度
- 在多个公开基准上刷新开源搜索代理最高成绩，同时优化资源使用与启动速度
  
## [prateek-chaubey/YTPro](https://github.com/prateek-chaubey/YTPro)

{{< shields path="github/stars/prateek-chaubey/YTPro" alt="Github Repo Stars" >}} {{< shields path="github/license/prateek-chaubey/YTPro" alt="License: `MIT`" >}} {{< shields path="github/languages/top/prateek-chaubey/YTPro" alt="Language: `Unknown`" >}}

{{< github-opengraph user="prateek-chaubey" repo="YTPro" alt="cover" >}}

YTPro 是一款兼容旧版安卓系统的 YouTube 客户端，集成多样实用功能以提升视频观看体验。

- 支持 Google Gemini 智能摘要与自定义提示词和模型
- 内置视频、短视频、缩略图及字幕下载器
- 广告屏蔽与跳过赞助商片段功能
- 背景音频播放及画中画模式，支持手势调节音量和亮度
- 显示不喜欢数、自定义收藏心形标记，无需登录账号保存视频
- 体积小于 50KB，界面简洁且几乎无内部依赖，自带自动更新机制
  
## [NVlabs/alpasim](https://github.com/NVlabs/alpasim)

{{< shields path="github/stars/NVlabs/alpasim" alt="Github Repo Stars" >}} {{< shields path="github/license/NVlabs/alpasim" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/NVlabs/alpasim" alt="Language: `Unknown`" >}}

{{< github-opengraph user="NVlabs" repo="alpasim" alt="cover" >}}

alpasim 是一个面向自动驾驶研究的模块化、轻量级且数据驱动的仿真平台。

- 提供真实感传感器数据模拟，包括神经渲染技术支持的高保真摄像头视角和环境噪声
- 支持端到端自动驾驶策略闭环测试，涵盖算法验证、安全分析、性能基准及调试
- 基于 Python 实现，具备高度可扩展性和灵活配置，通过 gRPC 接口实现组件替换与快速原型开发
- 采用微服务架构，实现分布式计算与多节点部署以满足横向扩展需求
- 内置多种先进驾驶策略模型，如 NVIDIA Alpamayo-R1 和 Latent TransFuser v6，便于对比评测
