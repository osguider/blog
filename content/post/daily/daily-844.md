## [twitter/the-algorithm](https://github.com/twitter/the-algorithm)

{{< shields path="github/stars/twitter/the-algorithm" alt="Github Repo Stars" >}} {{< shields path="github/license/twitter/the-algorithm" alt="License: `AGPL-3.0`" >}} {{< shields path="github/languages/top/twitter/the-algorithm" alt="Language: `Unknown`" >}}

![demo-picture-of-the-algorithm](https://static.osguider.com/subject/github/twitter/the-algorithm/33cb1f9cef22a1b4ab5e8e7ead3a58c7.png)

the-algorithm 是 Twitter 的推荐算法源代码。

- 提供服务和作业，负责在所有 Twitter 产品界面上提供推文和其他内容的动态。
- 包含多个核心组件，如推文服务、用户行为实时流、用户信号平台等。
- 采用多种模型进行社区检测、知识图谱嵌入及内容安全检测。
- 使用高性能机器学习模型服务框架，支持实时或批量生成聚合特征。
- 支持个性化推荐功能，包括为您时间线和推荐通知。
  
## [google/style-aligned](https://github.com/google/style-aligned)

{{< shields path="github/stars/google/style-aligned" alt="Github Repo Stars" >}} {{< shields path="github/license/google/style-aligned" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/google/style-aligned" alt="Language: `Unknown`" >}}

![demo-picture-of-style-aligned](https://static.osguider.com/subject/github/google/style-aligned/cf7862235cee7694a14ae91f83c0e6da.jpg)

style-aligned 是一个用于通过共享注意力生成风格对齐图像的官方代码。

- 支持使用 SDXL 生成风格对齐的图像。
- 提供从参考图像中提取样式并应用于新图像的功能。
- 能够结合 ControlNet 深度条件生成风格对齐和深度条件化的图像。
- 可用于使用 MultiDiffusion 生成全景风格对齐图片。
- 提供本地和在线演示，方便用户体验。
  
## [Tencent/cherry-markdown](https://github.com/Tencent/cherry-markdown)

{{< shields path="github/stars/Tencent/cherry-markdown" alt="Github Repo Stars" >}} {{< shields path="github/license/Tencent/cherry-markdown" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/Tencent/cherry-markdown" alt="Language: `Unknown`" >}}

{{< github-opengraph user="Tencent" repo="cherry-markdown" alt="cover" >}}

cherry-markdown 是一个轻量级的 JavaScript Markdown 编辑器，支持在浏览器或服务器（NodeJs）上运行。

- 开箱即用，支持常用的 Markdown 语法
- 易于扩展，可以快速进行二次开发
- 提供多种编辑模式，如实时预览、仅预览和无工具栏模式
- 内置安全机制，通过白名单过滤和 DomPurify 进行扫描过滤
- 支持图像缩放、对齐及引用功能，并可生成基于表格内容的图表
  
## [sh-lee-prml/HierSpeechpp](https://github.com/sh-lee-prml/HierSpeechpp)

{{< shields path="github/stars/sh-lee-prml/HierSpeechpp" alt="Github Repo Stars" >}} {{< shields path="github/license/sh-lee-prml/HierSpeechpp" alt="License: `MIT`" >}} {{< shields path="github/languages/top/sh-lee-prml/HierSpeechpp" alt="Language: `Unknown`" >}}

![demo-picture-of-HierSpeechpp](https://static.osguider.com/subject/github/sh-lee-prml/HierSpeechpp/e7c982661fb9e3ed1eaee39b31d77420.png)

HierSpeech++ 是一个基于层次变分推断的零-shot 语音合成系统，旨在弥合语义和声学表示之间的差距。

- 提供快速且强大的文本到语音（TTS）和声音转换（VC）功能
- 采用层次化框架显著提高合成语音的鲁棒性与表现力
- 实现了人类水平质量的零 -shot 语音合成
- 包含预训练模型，可用于多种数据集，如 LibriTTS
- 支持高效的 16kHz 到 48kHz 声频超分辨率处理
  
## [luciddreamer-cvlab/LucidDreamer](https://github.com/luciddreamer-cvlab/LucidDreamer)

{{< shields path="github/stars/luciddreamer-cvlab/LucidDreamer" alt="Github Repo Stars" >}} {{< shields path="github/license/luciddreamer-cvlab/LucidDreamer" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/luciddreamer-cvlab/LucidDreamer" alt="Language: `Unknown`" >}}

![demo-picture-of-LucidDreamer](https://static.osguider.com/subject/github/luciddreamer-cvlab/LucidDreamer/067ec6d850e0b73572ed2a64da4034c7.png)

LucidDreamer 是一个用于生成 3D 高斯点云场景的开源项目，旨在实现无领域限制的场景生成。

- 支持无领域限制的 3D 高斯点云场景生成
- 提供多种交互方式，包括本地和在线演示
- 兼容多个操作系统（Ubuntu 和 Windows）
- 可自定义输入图像和文本提示以指导场景生成
- 包含丰富的选项来调整相机路径和渲染设置
