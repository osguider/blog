## [KwaiKEG/KwaiAgents](https://github.com/KwaiKEG/KwaiAgents)

{{< shields path="github/stars/KwaiKEG/KwaiAgents" alt="Github Repo Stars" >}} {{< shields path="github/license/KwaiKEG/KwaiAgents" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/KwaiKEG/KwaiAgents" alt="Language: `Unknown`" >}}

![demo-picture-of-KwaiAgents](https://static.osguider.com/subject/github/KwaiKEG/KwaiAgents/42ccf7dd335248808a3190f40f72537a.png)

KwaiAgents 是一个基于大型语言模型（LLMs）的通用信息检索代理系统。

- 提供 KAgentSys-Lite，保留部分原系统功能的轻量版本。
- 包含一系列具有规划、反思和工具使用能力的大型语言模型（KAgentLMs）。
- 提供超过 20 万条与代理相关的指令微调数据（KAgentInstruct）。
- 拥有超过 3000 条人类编辑的自动评估数据，用于测试代理能力（KAgentBench）。
  
## [localsend/localsend](https://github.com/localsend/localsend)

{{< shields path="github/stars/localsend/localsend" alt="Github Repo Stars" >}} {{< shields path="github/license/localsend/localsend" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/localsend/localsend" alt="Language: `Unknown`" >}}

{{< github-opengraph user="localsend" repo="localsend" alt="cover" >}}

LocalSend 是一个开源的跨平台文件和消息共享应用，提供了 AirDrop 的替代方案。

- 支持在本地网络中安全分享文件和消息，无需互联网连接
- 使用 REST API 和 HTTPS 加密确保通信安全
- 跨平台兼容，包括 Android、iOS、macOS、Windows 和 Linux
- 快速可靠，不依赖第三方服务器或外部服务
  
## [radishes-music/radishes](https://github.com/radishes-music/radishes)

跨平台的无版权音乐平台，支持 windows / macos / linux / web。

- 支持浏览器、桌面级和移动端
- 提供丰富的音乐功能，包括每日推荐歌单和歌曲、歌单搜索、排行榜支持等

## [Carrot-collective/ERCF_v2](https://github.com/Carrot-collective/ERCF_v2)

{{< shields path="github/stars/Carrot-collective/ERCF_v2" alt="Github Repo Stars" >}} {{< shields path="github/license/Carrot-collective/ERCF_v2" alt="License: `GPL-3.0`" >}} {{< shields path="github/languages/top/Carrot-collective/ERCF_v2" alt="Language: `Unknown`" >}}

{{< github-opengraph user="Carrot-collective" repo="ERCF_v2" alt="cover" >}}

ERCF_v2 是一个为基于 Klipper 的 3D 打印机设计的可扩展多材料单元（MMU），旨在优化和改进 Voron ERCF MMU。

- 社区驱动项目，经过重大更新，提升了可靠性和简化了构造。
- 新增集成的纤维缓冲系统（ERCT）和切割选项（ERF）。
- 采用定制编码器设计，提高稳定性。
- 创新的纤维捕捉机制，避免缠绕问题。
- 提供详细手册与高质量 CAD 文件支持。
  
## [killme2008/aviatorscript](https://github.com/killme2008/aviatorscript)

{{< shields path="github/stars/killme2008/aviatorscript" alt="Github Repo Stars" >}} {{< shields path="github/license/killme2008/aviatorscript" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/killme2008/aviatorscript" alt="Language: `Unknown`" >}}

{{< github-opengraph user="killme2008" repo="aviatorscript" alt="cover" >}}

aviatorscript 是一门高性能、轻量级寄宿于 JVM 上的脚本语言。

- 支持多种基本数据类型，包括数字、字符串和布尔值。
- 函数是一等公民，支持闭包和函数式编程。
- 内置 bigint/decimal 类型用于大整数和高精度运算，并支持运算符重载。
- 完整的脚本语法，包含条件语句、循环语句及异常处理等功能。
- 轻量化模块系统，方便调用 Java 方法并完整支持 Java 脚本 API。
- 动态编译与执行，高性能且适用于 Android 等非标准 Java 平台。
- 提供丰富的定制选项，可作为安全沙箱或全功能语言使用。
