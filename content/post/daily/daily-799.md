## [rommapp/romm](https://github.com/rommapp/romm)

{{< shields path="github/stars/rommapp/romm" alt="Github Repo Stars" >}} {{< shields path="github/license/rommapp/romm" alt="License: `AGPL-3.0`" >}} {{< shields path="github/languages/top/rommapp/romm" alt="Language: `Unknown`" >}}

![demo-picture-of-romm](https://static.osguider.com/subject/github/rommapp/romm/4bc4ac2c68f31739c7f2b99c4961d6f3.png)

RomM 是一款美观且强大的自托管 ROM 管理器与网页端游戏玩家工具集。

- 支持从 IGDB 等平台自动扫描并丰富游戏库元数据
- 可从 SteamGridDB 获取定制封面图
- 集成 Retroachievements 显示成就系统
- 兼容 400 多种模拟平台的游戏识别
- 通过 EmulatorJS 和 RuffleRS 实现网页直接游玩经典主机 /PC 游戏
- 提供精细的权限控制系统便于分享收藏夹给好友查看访问
  
## [HotCakeX/Harden-Windows-Security](https://github.com/HotCakeX/Harden-Windows-Security)

{{< shields path="github/stars/HotCakeX/Harden-Windows-Security" alt="Github Repo Stars" >}} {{< shields path="github/license/HotCakeX/Harden-Windows-Security" alt="License: `MIT`" >}} {{< shields path="github/languages/top/HotCakeX/Harden-Windows-Security" alt="Language: `Unknown`" >}}

{{< github-opengraph user="HotCakeX" repo="Harden-Windows-Security" alt="cover" >}}

Harden-Windows-Security 是一个基于微软官方支持方法，旨在安全加固 Windows 系统的开源项目。

- 使用微软官方工具和功能进行系统加固，无需第三方软件介入，确保透明且无隐藏操作
- 持续更新以适配最新 Windows 版本和行业标准的安全措施
- 提供图形界面（GUI）和命令行界面（CLI）两种使用方式，方便个人及企业用户部署
- 兼顾个人、企业、政府及军事级别的安全需求，并附带详细说明与合理性分析文档
- 不移除或禁用任何微软推荐的重要功能，保持系统稳定性与兼容性
- 所有参考链接均来自权威官方渠道或知名机构网站，避免误导信息
  
## [coder/code-server](https://github.com/coder/code-server)

{{< shields path="github/stars/coder/code-server" alt="Github Repo Stars" >}} {{< shields path="github/license/coder/code-server" alt="License: `MIT`" >}} {{< shields path="github/languages/top/coder/code-server" alt="Language: `Unknown`" >}}

{{< github-opengraph user="coder" repo="code-server" alt="cover" >}}

code-server 是一款能够在任何设备上通过浏览器访问并使用 VS Code 的远程开发环境解决方案。

- 允许用户在任意机器上运行 VS Code，保持统一的开发环境体验
- 利用云服务器加速测试、编译和下载等资源密集型任务，减轻本地设备负担
- 支持低配置 Linux 服务器（1 GB RAM 和 2 vCPU）即可部署，通过 WebSockets 实现远程连接
- 提供多种安装方式，包括自动化脚本、一键云端部署及与 devcontainers 集成的灵活选项
- 针对团队协作提供 coder/coder 工具，实现集中管理和远程开发支持
  
## [lzhoang2801/OpCore-Simplify](https://github.com/lzhoang2801/OpCore-Simplify)

{{< shields path="github/stars/lzhoang2801/OpCore-Simplify" alt="Github Repo Stars" >}} {{< shields path="github/license/lzhoang2801/OpCore-Simplify" alt="License: `BSD-3-Clause`" >}} {{< shields path="github/languages/top/lzhoang2801/OpCore-Simplify" alt="Language: `Unknown`" >}}

{{< github-opengraph user="lzhoang2801" repo="OpCore-Simplify" alt="cover" >}}

OpCore-Simplify 是一款专注于自动化生成符合硬件配置的 OpenCore EFI 文件，简化 Hackintosh 系统搭建流程的工具。

- 根据完整硬件信息自动检测并添加所需 ACPI 补丁与驱动（kext），涵盖 CPU、GPU 及其他设备
- 支持从 Intel Nehalem 到最新 Arrow Lake，AMD Ryzen/Threadripper 等多代处理器，以及广泛 GPU 和 macOS 版本
- 集成 SSDTTime 常用补丁和自定义修复，解决内核崩溃、睡眠唤醒异常等问题，提高系统稳定性
- 自动更新 OpenCorePkg 与相关 kext，从官方 Dortania Builds 和 GitHub 获取最新版本保障兼容性
- 提供基于社区经验和个人调优的 EFI 配置选项，如 GPU ID 伪装、多核拓扑重构及 SIP 禁用等高级功能
  
## [microsoft/call-center-ai](https://github.com/microsoft/call-center-ai)

{{< shields path="github/stars/microsoft/call-center-ai" alt="Github Repo Stars" >}} {{< shields path="github/license/microsoft/call-center-ai" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/microsoft/call-center-ai" alt="Language: `Unknown`" >}}

{{< github-opengraph user="microsoft" repo="call-center-ai" alt="cover" >}}

call-center-ai 是基于 Azure 和 OpenAI GPT 构建的智能电话呼叫中心解决方案，支持通过 API 发起 AI 代理电话或直接从配置号码拨打机器人。

- 支持多语言、多语音风格的双向通话和短信交互，实时流式传输对话内容，实现断线续接与历史记录存储，提升客户体验并实现全天候服务
- 利用高性能 GPT-4.1 系列模型进行细致理解和安全合规的数据处理，支持领域专用术语、结构化理赔信息采集及自动任务生成，同时具备内容过滤与防越狱能力
- 提供高度可定制化功能，包括自定义提示词、特性开关、人类客服回退及通话录音，并计划引入自动回拨和 IVR 工作流，实现品牌专属声音塑造以增强识别度
- 基于 Azure 容器化无服务器架构部署，实现弹性伸缩与成本优化，无缝整合通信服务和认知资源，为快速迭代及多变负载提供安全稳定环境
