tailcall 是一个用于构建高性能 GraphQL 后端的开源解决方案。

- 提供高效的 GraphQL 运行时环境
- 支持多种安装方式，包括 NPM、Yarn 和 Docker
- 允许在同一文件中定义 GraphQL 模式和解析方法，简化开发流程
- 包含详细基准测试以比较不同 GraphQL 解决方案的性能
  
## [n8n-io/n8n](https://github.com/n8n-io/n8n)

{{< shields path="github/stars/n8n-io/n8n" alt="Github Repo Stars" >}} {{< shields path="github/license/n8n-io/n8n" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/n8n-io/n8n" alt="Language: `Unknown`" >}}

![demo-picture-of-n8n](https://static.osguider.com/subject/github/n8n-io/n8n/2f8b89d662fec25a9551151dd78ba31e.png)

n8n 是一个免费的、可自托管的工作流自动化工具，采用公平代码许可模式，旨在轻松实现不同服务之间的任务自动化。

- 提供 200 多个集成节点以支持各种工作流
- 允许用户添加自定义功能和逻辑
- 支持通过 Docker 运行和命令行使用
- 可与 LangChain 结合，实现 AI 驱动的功能
- 拥有活跃的社区支持与文档资源
