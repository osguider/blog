## [opencloud-eu/opencloud](https://github.com/opencloud-eu/opencloud)

{{< shields path="github/stars/opencloud-eu/opencloud" alt="Github Repo Stars" >}} {{< shields path="github/license/opencloud-eu/opencloud" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/opencloud-eu/opencloud" alt="Language: `Unknown`" >}}

{{< github-opengraph user="opencloud-eu" repo="opencloud" alt="cover" >}}

opencloud 是一个基于 Go 语言构建的云服务后端服务器项目，专注于提供核心后台服务功能。

- 使用 OpenID Connect 协议进行用户身份认证，支持外部身份提供者（如 Keycloak）及内嵌的 LibreGraph Connect 身份认证
- 不依赖数据库，所有数据均存储在文件系统中，默认根目录为用户主目录下的 .opencloud 文件夹
- 提供简便的构建和启动流程，可快速生成所需资源并运行本地测试实例
  
## [nickrunning/wechat-selkies](https://github.com/nickrunning/wechat-selkies)

{{< shields path="github/stars/nickrunning/wechat-selkies" alt="Github Repo Stars" >}} {{< shields path="github/license/nickrunning/wechat-selkies" alt="License: `MIT`" >}} {{< shields path="github/languages/top/nickrunning/wechat-selkies" alt="Language: `Unknown`" >}}

{{< github-opengraph user="nickrunning" repo="wechat-selkies" alt="cover" >}}

wechat-selkies 是一个基于 Selkies WebRTC 技术的 Linux 微信 /QQ 浏览器客户端，封装在 Docker 容器中，支持通过浏览器直接访问，无需本地安装微信或 QQ 客户端。

- 支持 AMD64 和 ARM64 架构，兼容主流 CPU
- 采用 Docker 容器化部署，实现环境隔离和便捷管理
- 支持数据和配置的持久化存储保障用户信息安全
- 完整中文字体及本地输入法支持，提升使用体验
- 提供图片复制、文件传输等实用功能，通过侧边栏面板操作方便
- 可选 GPU 硬件加速，提高性能表现
- 内置窗口切换悬浮窗增强多任务处理能力
- 配置灵活，可自动启动微信或 QQ 客户端，并支持自定义用户名与访问密码
- 适用于服务器部署和远程办公场景，通过 HTTPS 安全访问
