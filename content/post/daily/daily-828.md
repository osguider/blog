## [sinelaw/fresh](https://github.com/sinelaw/fresh)

{{< shields path="github/stars/sinelaw/fresh" alt="Github Repo Stars" >}} {{< shields path="github/license/sinelaw/fresh" alt="License: `GPL-2.0`" >}} {{< shields path="github/languages/top/sinelaw/fresh" alt="Language: `Unknown`" >}}

{{< github-opengraph user="sinelaw" repo="fresh" alt="cover" >}}

fresh 是一款面向终端的现代文本编辑器，兼具易用性、强大功能与高性能。

- 设计注重发现和易用，拥有原生用户界面、完整菜单系统及强大的命令面板，并支持全鼠标操作，实现从图形界面的平滑过渡
- 插件采用 TypeScript 编写，在沙箱化的 Deno 环境中安全运行，利用现代 JavaScript 生态实现灵活扩展且保证稳定性
- 极致性能优化，实现近零延迟响应，可流畅打开和编辑多达数 GB 的超大文件而无卡顿
- 功能全面涵盖文件管理（浏览器、多标签、自动还原）、编辑（多光标、智能缩进等）、搜索替换（增量搜索、多模式查找）、导航定位以及视图布局调整等常见需求
- 内置语言服务器协议支持，包括跳转定义、引用、高亮诊断及代码补全，提高编程效率与准确度

{{< shields path="github/stars/anthropics/claude-quickstarts" alt="Github Repo Stars" >}} {{< shields path="github/license/anthropics/claude-quickstarts" alt="License: `MIT`" >}} {{< shields path="github/languages/top/anthropics/claude-quickstarts" alt="Language: `Unknown`" >}}

{{< github-opengraph user="anthropics" repo="claude-quickstarts" alt="cover" >}}

claude-quickstarts 是一组示范项目，旨在帮助开发者快速构建基于 Claude API 的可部署应用。

- 提供多种预设的快速启动项目，包括客户支持代理、金融数据分析师、计算机操作演示和自主编码代理
- 展示如何利用 Claude 的自然语言理解与生成能力，实现智能客服和交互式数据分析等功能
- 支持通过环境变量配置 API 密钥，配套详细的安装与运行指南便于上手使用
  
## [microsoft/Foundry-Local](https://github.com/microsoft/Foundry-Local)

{{< shields path="github/stars/microsoft/Foundry-Local" alt="Github Repo Stars" >}} {{< shields path="github/license/microsoft/Foundry-Local" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/microsoft/Foundry-Local" alt="Language: `Unknown`" >}}

{{< github-opengraph user="microsoft" repo="Foundry-Local" alt="cover" >}}

Foundry-Local 是一款无需云端订阅即可在本地设备上运行生成式人工智能模型的工具，强调数据隐私和硬件性能优化。

- 支持直接在本地硬件上运行生成式 AI 模型，无需注册或连接 Azure 订阅
- 保证所有数据处理均在设备内完成，提升隐私与安全性
- 提供兼容 OpenAI 的 API 接口，方便将模型集成到现有应用中
- 利用 ONNX Runtime 和硬件加速技术自动优化模型性能，根据不同硬件（如 Nvidia CUDA GPU、Qualcomm NPU 或 CPU）下载对应版本
- 配备多语言 SDK（C#、Python、JavaScript），其中 C# SDK 完全自包含，无需依赖外部服务，可实现原生聊天补全和音频转录功能
  
## [cloudflare/vibesdk](https://github.com/cloudflare/vibesdk)

{{< shields path="github/stars/cloudflare/vibesdk" alt="Github Repo Stars" >}} {{< shields path="github/license/cloudflare/vibesdk" alt="License: `MIT`" >}} {{< shields path="github/languages/top/cloudflare/vibesdk" alt="Language: `Unknown`" >}}

{{< github-opengraph user="cloudflare" repo="vibesdk" alt="cover" >}}

vibesdk 是一个基于 Cloudflare 平台构建的开源 AI 驱动的全栈应用生成器，允许用户通过自然语言描述快速创建和部署定制化 Web 应用。

- 支持通过自然语言交互驱动 AI 代码生成，实现分阶段开发与智能错误修正
- 提供沙箱环境中的实时预览功能，确保应用效果即时可见
- 采用现代技术栈（React、TypeScript、Tailwind）自动生成前端代码
- 集成一键部署至 Cloudflare Workers 平台，简化上线流程
- 支持 GitHub 集成，可直接导出代码到版本库进行管理
- 构建于完整的 Cloudflare 开发生态，包括 Durable Objects、D1 数据库、多供应商大模型接口及 R2 存储等核心组件
  
## [KaijuEngine/kaiju](https://github.com/KaijuEngine/kaiju)

{{< shields path="github/stars/KaijuEngine/kaiju" alt="Github Repo Stars" >}} {{< shields path="github/license/KaijuEngine/kaiju" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/KaijuEngine/kaiju" alt="Language: `Unknown`" >}}

{{< github-opengraph user="KaijuEngine" repo="kaiju" alt="cover" >}}

kaiju 是一个基于 Go 语言和 Vulkan 的通用 2D/3D 游戏引擎，旨在通过现代系统级编程语言实现高性能与简洁性的结合。

- 支持 Windows、Linux、Android 平台，Mac 系统支持正在开发中
- 内置编辑器，方便游戏开发流程集成
- 性能显著优于主流游戏引擎（发布模式下比 Unity 快约 9 倍）
- 更快的构建速度和更低的内存占用
- 使用 Go 语言实现，使代码简单易懂且便于扩展，同时支持直接使用 Go 编写游戏逻辑
- 集成本地 AI（大规模语言模型）交互能力
- 垃圾回收机制设计合理，实现零堆分配运行，有效兼顾稳定性与性能
