## [roundcube/roundcubemail](https://github.com/roundcube/roundcubemail)

Roundcube 是一个基于浏览器的多语言 IMAP 邮件客户端，具有类似应用程序的用户界面。

- 提供完整的电子邮件客户端功能，包括 MIME 支持、地址簿、文件夹管理、消息搜索和拼写检查。
- 使用 PHP 编写，并支持 MariaDB、MySQL、PostgreSQL 或 SQLite 数据库。
- 具备插件 API，易于扩展，同时用户界面可通过皮肤完全自定义。
- 支持主流浏览器，如 Chrome, Firefox, Edge 等。
  
## [stan-smith/FossFLOW](https://github.com/stan-smith/FossFLOW)

{{< shields path="github/stars/stan-smith/FossFLOW" alt="Github Repo Stars" >}} {{< shields path="github/license/stan-smith/FossFLOW" alt="License: `MIT`" >}} {{< shields path="github/languages/top/stan-smith/FossFLOW" alt="Language: `Unknown`" >}}

{{< github-opengraph user="stan-smith" repo="FossFLOW" alt="cover" >}}

FossFLOW 是一款基于浏览器的开源渐进式网页应用，专注于创建美观的等距基础设施图表。

- 支持离线使用，运行在 React 和 Isoflow 库之上，实现流畅且高性能的图形绘制体验
- 多语言界面覆盖八种语言，包括英语、简体中文、西班牙语、葡萄牙语（巴西）、法语、印地语、孟加拉语和俄语，并自动识别用户语言偏好
- 改进连接工具，支持点击节点顺序连接和拖拽两种模式，提高连线操作的准确性与灵活性
- 允许导入自定义图标（PNG/JPG/SVG），并自动缩放及切换等距 3D 和平面 2D 显示方式，自定义资源可随项目保存跨设备同步
- 提供服务器端存储功能，通过 Docker 部署实现多设备访问与数据持久化，同时具备重名覆盖保护机制
- 丰富交互优化，包括可配置快捷键、多样化平移控制方式以及连接箭头显示切换，提升用户操作效率与体验

## [Mailu/Mailu](https://github.com/Mailu/Mailu)

{{< shields path="github/stars/Mailu/Mailu" alt="Github Repo Stars" >}} {{< shields path="github/license/Mailu/Mailu" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/Mailu/Mailu" alt="Language: `Unknown`" >}}

![demo-picture-of-Mailu](https://static.osguider.com/subject/github/Mailu/Mailu/7a2d3bba70526e3853d4c36172d0a00f.png)

Mailu 是一个简单而功能全面的邮件服务器，采用一组 Docker 镜像构建。

- 标准电子邮件服务器，支持 IMAP、IMAP+、SMTP 和提交，并为客户端提供自动配置文件
- 高级电子邮件功能，包括别名、域别名、自定义路由和附件全文搜索
- 提供网页访问、多种 Web 邮件和管理界面
- 用户功能包括别名、自动回复、自动转发和账户管理
- 管理员特性，如全局管理员公告、按域委派及配额设置
- 安全性强，支持强制 TLS、安全传输协议（DANE）、MTA-STS 和 Letsencrypt，以及 DKIM 签名、防病毒扫描等
- 反垃圾邮件机制，包括自学习灰名单技术以及 DMARC 和 SPF 防伪措施
- 所有组件均为自由开源软件，无跟踪器包含
