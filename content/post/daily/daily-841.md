## [vectara/hallucination-leaderboard](https://github.com/vectara/hallucination-leaderboard)

{{< shields path="github/stars/vectara/hallucination-leaderboard" alt="Github Repo Stars" >}} {{< shields path="github/license/vectara/hallucination-leaderboard" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/vectara/hallucination-leaderboard" alt="Language: `Unknown`" >}}

![demo-picture-of-hallucination-leaderboard](https://static.osguider.com/subject/github/vectara/hallucination-leaderboard/610fb379c8161f4740c181a5e5b728f8.png)

hallucination-leaderboard 是一个比较大型语言模型（LLM）在总结短文档时产生幻觉的表现的排行榜。

- 使用 Vectara 的 Hughes 幻觉评估模型计算，定期更新。
- 提供不同 LLM 在生成摘要时的幻觉率、事实一致性率和回答率等数据。
- 允许用户查看基于旧版本评估模型的数据。
- 在 Hugging Face 上也可以访问该排行榜。
  
## [eylonmiz/react-agent](https://github.com/eylonmiz/react-agent)

{{< shields path="github/stars/eylonmiz/react-agent" alt="Github Repo Stars" >}} {{< shields path="github/license/eylonmiz/react-agent" alt="License: `MIT`" >}} {{< shields path="github/languages/top/eylonmiz/react-agent" alt="Language: `Unknown`" >}}

![demo-picture-of-react-agent](https://static.osguider.com/subject/github/eylonmiz/react-agent/709a19251f0a6d94d2219bf4c0c80759.jpg)

react-agent 是一个开源的 React.js 自主 LLM 代理，利用 GPT-4 语言模型根据用户故事生成和组合 React 组件。

- 从用户故事生成 React 组件
- 从现有组件组合生成新的 React 组件
- 使用本地设计系统来创建 React 组件
- 基于 Atomic Design 原则构建
- 支持使用多种技术栈，包括 React、TailwindCSS、Typescript 等
  
## [gatsbyjs/gatsby](https://github.com/gatsbyjs/gatsby)

{{< shields path="github/stars/gatsbyjs/gatsby" alt="Github Repo Stars" >}} {{< shields path="github/license/gatsbyjs/gatsby" alt="License: `MIT`" >}} {{< shields path="github/languages/top/gatsbyjs/gatsby" alt="Language: `Unknown`" >}}

{{< github-opengraph user="gatsbyjs" repo="gatsby" alt="cover" >}}

Gatsby 是一个基于 React 的开源框架，旨在帮助开发者构建快速、高效且安全的网站和应用。

- 支持从多种数据源加载数据，包括 Markdown 文件、无头 CMS 和 REST 或 GraphQL API。
- 提供动态渲染与静态网站生成的结合，适用于各种类型的网站。
- 允许按页面选择不同的渲染选项，如延迟静态生成和服务器端渲染，以优化性能。
- 自动化代码分割、图像优化等功能，提高网站性能，无需手动调整。
- 使用现代技术栈（React 和 GraphQL），确保团队工作流程一致性。
- 可以通过 CDN 托管，大幅降低成本，并支持免费托管。
  
## [THUDM/AgentTuning](https://github.com/THUDM/AgentTuning)

{{< shields path="github/stars/THUDM/AgentTuning" alt="Github Repo Stars" >}} {{< shields path="github/license/THUDM/AgentTuning" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/THUDM/AgentTuning" alt="Language: `Unknown`" >}}

![demo-picture-of-AgentTuning](https://static.osguider.com/subject/github/THUDM/AgentTuning/e813e3cfb7403dfbaae15a8ec5876897.svg)

AgentTuning 是一个旨在通过多种代理任务的交互轨迹来对大型语言模型（LLMs）进行指令调优的项目。

- 首个尝试使用多个代理任务的交互轨迹进行 LLMs 的指令调优
- 提供了高质量的数据集 AgentInstruct，包含 1,866 个互动示例，涵盖六种真实世界任务
- 模型 AgentLM 在多样化场景中表现出强大的泛化能力和语言理解能力
- 严格过滤数据以确保高质量，并避免数据泄露风险
- 支持 Docker 部署，便于快速评估和实例运行
  
## [JetBrains/intellij-platform-plugin-template](https://github.com/JetBrains/intellij-platform-plugin-template)
