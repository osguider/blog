## [booklore-app/booklore](https://github.com/booklore-app/booklore)

{{< shields path="github/stars/booklore-app/booklore" alt="Github Repo Stars" >}} {{< shields path="github/license/booklore-app/booklore" alt="License: `GPL-3.0`" >}} {{< shields path="github/languages/top/booklore-app/booklore" alt="Language: `Unknown`" >}}

![demo-picture-of-booklore](https://static.osguider.com/subject/github/adityachandelgit/BookLore/c964fa6e1afe7dcfe343341ea3d87baf.gif)

booklore 是一个自托管的多用户数字图书馆管理系统，支持智能书架、自动元数据获取及多设备同步。

- 智能组织：通过自定义书架、智能排序和强大筛选功能，实现快速精准的图书查找
- 多用户管理：支持添加多个用户并设定细粒度权限，保障内容访问与控制安全
- 设备同步：无缝对接 Kobo 和 KOReader，实现阅读进度跨设备同步与 EPUB 自动转换为 KEPUB
- 自动元数据抓取：从 Goodreads、Amazon 等平台自动获取丰富图书信息，提升资料完整性
- 动态魔法书架：基于规则实时更新的动态集合，如同为藏书打造智能播放列表
- 文件导入便捷化：通过 BookDrop 功能实现文件夹内批量导入，无需手动操作繁琐流程
- OPDS 支持及内置阅读器：兼容主流阅读应用无线下载，同时提供 PDF、EPUB 和漫画格式本地浏览体验
  
## [fujiapple852/trippy](https://github.com/fujiapple852/trippy)

![demo-picture-of-trippy](https://static.osguider.com/subject/github/fujiapple852/trippy/e1a043bb529d90077bdb3972fa9689ee.svg)

trippy 是一个集成了 traceroute 和 ping 功能的网络诊断工具，旨在帮助分析网络问题。

- 支持多平台运行，包括 Linux、BSD、macOS 和 Windows
- 提供多种安装方式，如包管理器安装、预编译二进制文件及源码安装
- 结合了多个开源项目和库的设计思想，提升功能丰富性和稳定性
- 提供详细文档支持，包括快速入门指南、使用示例及命令参考等
- 支持无管理员权限运行，并提供相关配置指导
- 集成 IP 到 ASN 的映射服务，有助于更深入的网络路径分析
  
## [frappe/hrms](https://github.com/frappe/hrms)

{{< shields path="github/stars/frappe/hrms" alt="Github Repo Stars" >}} {{< shields path="github/license/frappe/hrms" alt="License: `GPL-3.0`" >}} {{< shields path="github/languages/top/frappe/hrms" alt="Language: `Unknown`" >}}

![demo-picture-of-hrms](https://static.osguider.com/subject/github/frappe/hrms/7b5b0bd08161d1f0033b9594d0ec7cb6.png)

hrms 是一款开源的现代化人力资源和薪资管理软件。

- 提供完整的人力资源管理系统（HRMS），涵盖员工生命周期管理、入职、晋升调动及离职面谈等全流程。
- 支持请假与考勤功能，包含区域节假日自动导入、地理位置打卡以及请假余额和考勤报表统计。
- 管理员工费用报销与预支，支持多级审批流程，并可无缝集成 ERPNext 财务模块。
- 绩效管理模块帮助设定目标并对齐关键结果领域（KRA），支持自我评估及简化绩效考核周期操作。
- 薪资与税务处理功能完善，包括工资结构配置、个税阶梯设置、多种支付方式及工资单详细展示。
- 配备移动应用，实现随时随地申请审批请假，打卡签到，以及访问员工资料等便捷操作。
  
## [SkyworkAI/SkyReels-V2](https://github.com/SkyworkAI/SkyReels-V2)

{{< shields path="github/stars/SkyworkAI/SkyReels-V2" alt="Github Repo Stars" >}} {{< shields path="github/license/SkyworkAI/SkyReels-V2" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/SkyworkAI/SkyReels-V2" alt="Language: `Unknown`" >}}

![demo-picture-of-SkyReels-V2](https://static.osguider.com/subject/github/SkyworkAI/SkyReels-V2/dfff6225cc325b6346e93bc716bd5349.png)

SkyReels-V2 是一个无限长度影片生成模型，采用自回归扩散强制（AutoRegressive Diffusion-Forcing）架构，实现了公开可用视频生成模型中的最先进性能。

- 支持无限长度的视频自动回归生成，突破传统视频时长限制
- 同时支持文本到视频（Text-to-Video）和图像到视频（Image-to-Video）的合成任务
- 提供多种分辨率版本，包括 540P 和 720P，以适应不同应用需求
- 开源提供模型权重及推理代码，便于研究与开发使用
- 包含多款相关子项目，如可控视觉元素组装的视频生成框架、肖像动画等功能模块
- 支持单 GPU 及多 GPU 推理模式，提高灵活性与效率
  
## [basecamp/fizzy](https://github.com/basecamp/fizzy)

{{< shields path="github/stars/basecamp/fizzy" alt="Github Repo Stars" >}} {{< shields path="github/license/basecamp/fizzy" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/basecamp/fizzy" alt="Language: `Unknown`" >}}

{{< github-opengraph user="basecamp" repo="fizzy" alt="cover" >}}

fizzy 是 37signals 开发的一款专注于问题和创意管理的看板追踪工具，旨在提供更符合现代需求的看板体验。

- 支持 SQLite 和 MySQL 数据库，可通过环境变量灵活切换
- 内置开发环境配置脚本及测试流程，支持快速单元测试与完整持续集成测试
- 集成邮件预览功能，并可自动打开发送邮件以便调试
- 采用 VAPID 标准实现浏览器推送通知，需要生成并配置密钥对
- 提供配套 SaaS 插件 fizzy-saas，用于连接计费系统及生产部署设置
