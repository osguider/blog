## [NVIDIA/Stable-Diffusion-WebUI-TensorRT](https://github.com/NVIDIA/Stable-Diffusion-WebUI-TensorRT)

{{< shields path="github/stars/NVIDIA/Stable-Diffusion-WebUI-TensorRT" alt="Github Repo Stars" >}} {{< shields path="github/license/NVIDIA/Stable-Diffusion-WebUI-TensorRT" alt="License: `MIT`" >}} {{< shields path="github/languages/top/NVIDIA/Stable-Diffusion-WebUI-TensorRT" alt="Language: `Unknown`" >}}

{{< github-opengraph user="NVIDIA" repo="Stable-Diffusion-WebUI-TensorRT" alt="cover" >}}

Stable-Diffusion-WebUI-TensorRT 是一个为稳定扩散 Web 界面提供 TensorRT 扩展的项目，旨在优化 NVIDIA RTX GPU 上的性能。

- 支持多种版本的稳定扩散，包括 1.5、2.1、SDXL 和 SDXL Turbo。
- 通过生成优化引擎来提高图像生成速度。
- 提供静态和动态引擎选项，以适应不同分辨率和批处理大小需求。
- 支持 LoRA / LyCORIS 检查点转换为 TensorRT 格式以便使用。
- 详细安装与使用说明，确保用户能够顺利配置与运行。
  
## [flipped-aurora/gin-vue-admin](https://github.com/flipped-aurora/gin-vue-admin)

{{< shields path="github/stars/flipped-aurora/gin-vue-admin" alt="Github Repo Stars" >}} {{< shields path="github/license/flipped-aurora/gin-vue-admin" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/flipped-aurora/gin-vue-admin" alt="Language: `Unknown`" >}}

![demo-picture-of-gin-vue-admin](https://static.osguider.com/subject/github/flipped-aurora/gin-vue-admin/78e04b7abe1abfd3d13afe3f5195f1bb.jpg)

gin-vue-admin 是一个基于 Vite、Vue3 和 Gin 的全栈前后端分离开发平台，支持 AI 辅助的基础开发。

- 支持 TS 和 JS 混用
- 集成 JWT 鉴权和权限管理
- 提供动态路由和动态菜单功能
- 包含表单生成器和代码生成器
- 实现多点登录拦截及资源权限控制
- 具备上传下载及可配置的导入导出功能

- 集成知名软件如 Postfix 和 Dovecot

- 包含所有已发布的斯诺登泄露文件，便于公众获取信息
