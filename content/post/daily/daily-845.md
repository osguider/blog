## [SqueezeAILab/LLMCompiler](https://github.com/SqueezeAILab/LLMCompiler)

{{< shields path="github/stars/SqueezeAILab/LLMCompiler" alt="Github Repo Stars" >}} {{< shields path="github/license/SqueezeAILab/LLMCompiler" alt="License: `MIT`" >}} {{< shields path="github/languages/top/SqueezeAILab/LLMCompiler" alt="Language: `Unknown`" >}}

![demo-picture-of-LLMCompiler](https://static.osguider.com/subject/github/SqueezeAILab/LLMCompiler/ecf6e538ef3282b49e1442659b3bac57.png)

LLMCompiler 是一个用于并行函数调用的 LLM 编译器框架。

- 自动识别可并行执行和相互依赖的任务，提高效率
- 支持开源和闭源模型，如 LLaMA 和 OpenAI 的 GPT 模型
- 通过将问题分解为多个可并行执行的任务，减少延迟和成本
- 用户可以指定工具及示例，自动计算优化后的函数调用编排
- 在多种任务中展示出一致的延迟加速、成本节省和准确性提升

## [GitSquared/edex-ui](https://github.com/GitSquared/edex-ui)

{{< shields path="github/stars/GitSquared/edex-ui" alt="Github Repo Stars" >}} {{< shields path="github/license/GitSquared/edex-ui" alt="License: `GPL-3.0`" >}} {{< shields path="github/languages/top/GitSquared/edex-ui" alt="Language: `Unknown`" >}}

![demo-picture-of-edex-ui](https://static.osguider.com/subject/github/GitSquared/edex-ui/3fa8b281ac9b4c9b23b8a355524947b4.png)

eDEX-UI 是一个跨平台的可定制科幻终端模拟器，具有先进的监控和触摸屏支持。

- 全功能终端模拟器，支持标签、颜色、鼠标事件以及 curses 和类似应用程序。
- 实时系统（CPU、RAM、交换区、进程）和网络（GeoIP、活动连接、传输速率）监控。
- 完全支持触摸显示，包括屏幕键盘。
- 目录查看器跟随终端当前工作目录。
- 高级自定义选项，包括主题、自定义键盘布局和 CSS 注入。
  
## [tldraw/make-real-starter](https://github.com/tldraw/make-real-starter)

{{< shields path="github/stars/tldraw/make-real-starter" alt="Github Repo Stars" >}} {{< shields path="github/license/tldraw/make-real-starter" alt="License: `AGPL-3.0`" >}} {{< shields path="github/languages/top/tldraw/make-real-starter" alt="Language: `Unknown`" >}}

{{< github-opengraph user="tldraw" repo="make-real-starter" alt="cover" >}}

make-real-starter 是一个用于创建无限画布风格应用的模板项目。

- 提供了使用 tldraw 库构建无限画布体验的基础。
- 支持将手绘 UI 草图转换为 HTML 文件，利用 OpenAI 的 GPT-4 Vision。
- 允许用户在画布上与生成的内容进行交互，并可迭代修改结果。
