## [carthage-software/mago](https://github.com/carthage-software/mago)

{{< shields path="github/stars/carthage-software/mago" alt="Github Repo Stars" >}} {{< shields path="github/license/carthage-software/mago" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/carthage-software/mago" alt="Language: `Unknown`" >}}

{{< github-opengraph user="carthage-software" repo="mago" alt="cover" >}}

mago 是一个基于 Rust 开发的 PHP 工具链，旨在帮助开发者编写更高质量的代码。

- 速度极快，利用 Rust 提供卓越性能
- 支持代码 lint 检查，可自定义规则识别潜在问题
- 提供深度静态分析以捕捉类型错误和漏洞
- 自动修复多种 lint 问题，提高开发效率
- 自动格式化代码，保持风格一致性和最佳实践
- 语义检查确保代码正确性与健壮性
- 支持抽象语法树（AST）可视化，方便理解代码结构
  
## [sapientinc/HRM](https://github.com/sapientinc/HRM)

{{< shields path="github/stars/sapientinc/HRM" alt="Github Repo Stars" >}} {{< shields path="github/license/sapientinc/HRM" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/sapientinc/HRM" alt="Language: `Unknown`" >}}

{{< github-opengraph user="sapientinc" repo="HRM" alt="cover" >}}

HRM 是一个基于层级递归架构的推理模型，旨在通过模拟人脑多时尺度处理机制，实现高效且稳定的复杂目标导向序列推理。

- 采用双模块设计：高层模块负责缓慢抽象规划，低层模块执行快速细节计算，实现单次前向传递中的连续推理
- 参数量仅 2700 万，在无预训练和无链式思维数据支持下，仅用 1000 个样本即可完成复杂数独和大规模迷宫路径寻找等任务
- 在人工智能通用能力基准 ARC 上表现优异，超越许多参数更多、上下文窗口更长的大型模型
- 结合 CUDA 扩展与 FlashAttention 优化，可在现代 GPU 设备上实现较快训练速度与资源利用效率
  
## [astral-sh/ty](https://github.com/astral-sh/ty)

{{< shields path="github/stars/astral-sh/ty" alt="Github Repo Stars" >}} {{< shields path="github/license/astral-sh/ty" alt="License: `MIT`" >}} {{< shields path="github/languages/top/astral-sh/ty" alt="Language: `Unknown`" >}}

{{< github-opengraph user="astral-sh" repo="ty" alt="cover" >}}

ty 是一个用 Rust 编写的极其快速的 Python 类型检查器和语言服务器。

- 性能显著优于同类工具，速度比 mypy 和 Pyright 快 10 到 100 倍
- 提供详尽且富有上下文信息的诊断报告，支持配置规则级别、文件覆盖和注释抑制
- 支持部分类型标注与重声明，便于逐步采用并兼容现有代码库
- 集成语言服务器功能，包括代码导航、补全、自动导入、内联提示及悬停帮助等编辑器辅助特性
- 精细增量分析设计，实现编辑时快速响应与更新，提高开发效率
- 支持多种主流编辑器集成，如 VS Code、PyCharm 和 Neovim 等
- 拥有先进类型系统特性，如交叉类型、一流的类型缩小以及复杂可达性分析
  
## [firecrawl/open-lovable](https://github.com/firecrawl/open-lovable)

{{< shields path="github/stars/firecrawl/open-lovable" alt="Github Repo Stars" >}} {{< shields path="github/license/firecrawl/open-lovable" alt="License: `MIT`" >}} {{< shields path="github/languages/top/firecrawl/open-lovable" alt="Language: `Unknown`" >}}

{{< github-opengraph user="firecrawl" repo="open-lovable" alt="cover" >}}

open-lovable 是一个通过与 AI 对话，瞬间克隆并重建任意网站为现代 React 应用的工具。

- 支持多种大型语言模型（LLM）API 接入，包括 Anthropic、OpenAI、Gemini 和 Groq
- 提供快速编辑功能，通过 Morph LLM 加速应用修改流程
- 集成 Vercel 或 E2B 沙箱环境，实现云端部署和测试的灵活选择
- 以命令行方式简便安装和启动，适合开发者快速上手使用
