## [karpathy/nn-zero-to-hero](https://github.com/karpathy/nn-zero-to-hero)

{{< shields path="github/stars/karpathy/nn-zero-to-hero" alt="Github Repo Stars" >}} {{< shields path="github/license/karpathy/nn-zero-to-hero" alt="License: `MIT`" >}} {{< shields path="github/languages/top/karpathy/nn-zero-to-hero" alt="Language: `Unknown`" >}}

{{< github-opengraph user="karpathy" repo="nn-zero-to-hero" alt="cover" >}}

nn-zero-to-hero 是一个从基础开始，逐步深入讲解神经网络原理与实践的教学课程。

- 通过系列视频和配套 Jupyter 笔记本，带领学习者编码并训练神经网络模型
- 涵盖反向传播、语言建模、多层感知机（MLP）、激活函数、梯度分析及批量归一化等核心技术
- 强调理论与实操结合，通过手动实现反向传播加深对优化过程的理解
- 逐步引入复杂结构，如基于字符的大型语言模型和树状深层网络架构
  
## [inventree/InvenTree](https://github.com/inventree/InvenTree)

{{< shields path="github/stars/inventree/InvenTree" alt="Github Repo Stars" >}} {{< shields path="github/license/inventree/InvenTree" alt="License: `MIT`" >}} {{< shields path="github/languages/top/inventree/InvenTree" alt="Language: `Unknown`" >}}

{{< github-opengraph user="inventree" repo="InvenTree" alt="cover" >}}

InvenTree 是一个开源的库存管理系统。

- 提供强大的低级库存控制和零件跟踪功能
- 基于 Python/Django 构建，拥有网页管理界面和 REST API，方便与外部应用集成
- 支持插件扩展，实现自定义应用和功能增强
- 兼容多种数据库（PostgreSQL、MySQL、SQLite 等）及现代前端技术栈（React 等）
- 多种部署方式，包括 Docker 容器化部署和单行安装脚本支持多平台安装
- 配套移动应用支持 Android 及 iOS，实现随时随地的库存访问与操作
- 注重安全性并提供详细的安全政策文档，同时社区贡献翻译支持多语言环境
  
## [lingodotdev/lingo.dev](https://github.com/lingodotdev/lingo.dev)

{{< shields path="github/stars/lingodotdev/lingo.dev" alt="Github Repo Stars" >}} {{< shields path="github/license/lingodotdev/lingo.dev" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/lingodotdev/lingo.dev" alt="Language: `Unknown`" >}}

{{< github-opengraph user="lingodotdev" repo="lingo.dev" alt="cover" >}}

lingo.dev 是一个开源的、基于人工智能的大型语言模型（LLM）驱动的国际化 (i18n) 工具包，旨在实现即时本地化。

- 提供编译器中间件，可在构建时自动将 React 应用多语言化，无需修改现有组件
- 命令行工具支持一键翻译代码和内容，支持多种格式如 JSON、YAML 和 Markdown，并具备缓存机制只重新翻译变更部分
- 集成 CI/CD 流程，实现每次代码推送时自动提交翻译并创建拉取请求，保持项目持续多语言更新
- SDK 支持动态内容的实时按需翻译，适用于聊天、用户评论等场景
- 支持自定义大型语言模型或使用内置 Lingo.dev 本地化引擎，提高灵活性与扩展性
  
## [github/copilot-cli](https://github.com/github/copilot-cli)

{{< shields path="github/stars/github/copilot-cli" alt="Github Repo Stars" >}} {{< shields path="github/license/github/copilot-cli" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/github/copilot-cli" alt="Language: `Unknown`" >}}

{{< github-opengraph user="github" repo="copilot-cli" alt="cover" >}}

copilot-cli 是一个将 GitHub Copilot 的 AI 编码助手功能直接集成到终端的工具。

- 允许用户通过自然语言与 AI 进行交互，完成代码编写、调试和理解任务
- 深度整合 GitHub 工作流，可访问仓库、问题和拉取请求
- 支持复杂任务的规划与执行，具备智能协作能力
- 提供基于 MCP（多模型代理）服务器的扩展性支持，自定义功能更灵活
- 用户对所有操作拥有完全控制权，每一步都需明确确认后才执行
- 跨平台支持 Linux、macOS 和 Windows，并依赖 Node.js 环境运行
  
## [NARKOZ/hacker-scripts](https://github.com/NARKOZ/hacker-scripts)

{{< shields path="github/stars/NARKOZ/hacker-scripts" alt="Github Repo Stars" >}} {{< shields path="github/license/NARKOZ/hacker-scripts" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/NARKOZ/hacker-scripts" alt="Language: `Unknown`" >}}

{{< github-opengraph user="NARKOZ" repo="hacker-scripts" alt="cover" >}}

hacker-scripts 是一组基于真实故事的自动化脚本集合，展示了一个热爱终端和编程的构建工程师如何通过脚本简化日常任务。

- 包含多种实用且幽默的自动化脚本，如发送随机借口短信、监控邮件并自动回滚数据库、定时发送请假邮件等。
- 利用 cron 作业实现定时执行，提高工作效率和生活便利性。
- 脚本涵盖 Shell 及 Ruby 语言，支持 Twilio 短信服务和 Gmail 邮箱操作。
- 创意十足，例如远程控制网络咖啡机按需冲泡咖啡，体现极致的自动化与趣味结合。
