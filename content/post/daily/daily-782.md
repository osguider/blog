## [jwohlwend/boltz](https://github.com/jwohlwend/boltz)

{{< shields path="github/stars/jwohlwend/boltz" alt="Github Repo Stars" >}} {{< shields path="github/license/jwohlwend/boltz" alt="License: `MIT`" >}} {{< shields path="github/languages/top/jwohlwend/boltz" alt="Language: `Unknown`" >}}

![demo-picture-of-boltz](https://static.osguider.com/subject/github/jwohlwend/boltz/bba0dacc5c3944e01f5be4b56a41b3ec.png)

Boltz 是一个用于生物分子相互作用预测的模型家族。

- Boltz-1 是第一个完全开源的模型，接近 AlphaFold3 的准确性。
- Boltz-2 超越了前者，通过联合建模复杂结构和结合亲和力，实现更精确的分子设计。
- 首个深度学习模型在速度上比基于物理的方法快 1000 倍，使早期药物发现中的计算筛选变得可行。
- 所有代码和权重均在 MIT 许可证下提供，适用于学术及商业用途。
  
## [EvolutionAPI/evolution-api](https://github.com/EvolutionAPI/evolution-api)

{{< shields path="github/stars/EvolutionAPI/evolution-api" alt="Github Repo Stars" >}} {{< shields path="github/license/EvolutionAPI/evolution-api" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/EvolutionAPI/evolution-api" alt="Language: `Unknown`" >}}

{{< github-opengraph user="EvolutionAPI" repo="evolution-api" alt="cover" >}}

evolution-api 是一个开源的多平台消息集成 API，最初基于 Baileys 库实现 WhatsApp 控制功能，现已扩展支持包括官方 WhatsApp Business API、Typebot、Chatwoot、Dify 和 OpenAI 等多种服务。

- 支持基于 Baileys 的免费 WhatsApp Web API 和 Meta 官方的云端 WhatsApp Business API 两种连接方式
- 提供丰富的第三方集成，如构建对话机器人（Typebot）、客户服务系统（Chatwoot）、消息队列事件接收（RabbitMQ、Amazon SQS）及 AI 能力扩展（OpenAI）
- 具备轻量级版本 Evolution API Lite，适合微服务环境下高效简洁使用
- 支持媒体文件存储到 Amazon S3 或 Minio，提高数据管理灵活性
- 实施非敏感数据遥测以优化产品性能与用户体验
  
## [coze-dev/coze-studio](https://github.com/coze-dev/coze-studio)

{{< shields path="github/stars/coze-dev/coze-studio" alt="Github Repo Stars" >}} {{< shields path="github/license/coze-dev/coze-studio" alt="License: `Apache-2.0`" >}} {{< shields path="github/languages/top/coze-dev/coze-studio" alt="Language: `Unknown`" >}}

{{< github-opengraph user="coze-dev" repo="coze-studio" alt="cover" >}}

coze-studio 是一个集成化的 AI 代理开发平台，提供全方位的可视化工具，简化了 AI 代理的创建、调试和部署流程。

- 支持最新大型模型及多种开发模式与框架，打造便捷高效的 AI 代理开发环境
- 提供包括提示词（prompt）、检索增强生成（RAG）、插件、工作流等核心技术支持
- 内置完整应用模板和构建框架，实现低成本快速构建多样 AI 代理及业务逻辑定制
- 基于微服务架构与领域驱动设计原则，高性能、高扩展性且易定制的后端框架
- 前端采用 React + TypeScript 实现，可通过无代码或低代码方式进行视觉设计和调试
- 支持模型管理、多资源管理（插件、知识库、数据库等）以及开放 API 和 SDK 集成能力
  
## [saagarjha/Ensemble](https://github.com/saagarjha/Ensemble)

{{< shields path="github/stars/saagarjha/Ensemble" alt="Github Repo Stars" >}} {{< shields path="github/license/saagarjha/Ensemble" alt="License: `AGPL-3.0`" >}} {{< shields path="github/languages/top/saagarjha/Ensemble" alt="Language: `Unknown`" >}}

![demo-picture-of-Ensemble](https://static.osguider.com/subject/github/saagarjha/Ensemble/6c5e4eaf945b70141c1453fd34d551a5.png)

Ensemble 是一个将 Mac 窗口无线投射到 visionOS 的工具。

- 允许用户直接在 visionOS 中移动、调整大小和交互 Mac 窗口
- 无线连接，避免了分辨率限制和扁平显示的局限性
- 简单易用，可以通过 Xcode 自行构建，无需第三方依赖
- 当前处于 alpha 阶段，适合测试使用，但可能会出现崩溃或故障

{{< github-opengraph user="Rapha-Borges" repo="oke-free" alt="cover" >}}

oke-free 是一个简单易用的工具，旨在帮助用户免费创建和管理自己的 Kubernetes 集群。

- 提供在 Oracle Cloud 上快速创建 Kubernetes 集群的解决方案
- 使用 Terraform 进行集群配置，简化部署流程
- 支持多种操作系统，包括 GNU/Linux 和 Windows
- 包含详细的安装指南和步骤，以确保顺利设置环境
- 允许用户利用免费的云资源，同时提供升级选项以满足更高需求
