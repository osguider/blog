sudo 是 Windows 上的 sudo。
sudo for Windows 允许用户在非提升权限的终端窗口中直接运行提升权限命令。主要功能和优势包括：

- 可以在 Windows 11 版本 26045 及更高版本上使用
- 提供了类似于 Linux 中 sudo 的概念，但是是针对 Windows 平台进行实现
- 可以通过 PowerShell 脚本 sudo.ps1 来更友好地使用 sudo 命令
