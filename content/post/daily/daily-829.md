wechat-format 是一个微信公众号排版编辑器，旨在将 Markdown 转换为微信特制的 HTML。

- 能较好地排版
- 解决 ul 和 ol 元素样式渲染时被微信编辑器重置的问题

## [evilsocket/legba](https://github.com/evilsocket/legba)

{{< shields path="github/stars/evilsocket/legba" alt="Github Repo Stars" >}} {{< shields path="github/license/evilsocket/legba" alt="License: `NOASSERTION`" >}} {{< shields path="github/languages/top/evilsocket/legba" alt="Language: `Unknown`" >}}

{{< github-opengraph user="evilsocket" repo="legba" alt="cover" >}}

legba 是一个多协议的凭证暴力破解器/密码喷洒工具和枚举器，使用 Rust 和 Tokio 异步运行时构建，以实现更好的性能和稳定性，同时消耗比类似工具更少的资源。

- 支持多种协议，包括 AMQP、Cassandra、DNS 子域枚举、FTP、HTTP 认证等
- 提供高效的异步处理，显著提高了执行速度
- 具有较低的资源消耗，相比其他工具更加节省系统资源
- 包含详细文档，便于用户快速上手
  
## [wechatsync/Wechatsync](https://github.com/wechatsync/Wechatsync)

{{< shields path="github/stars/wechatsync/Wechatsync" alt="Github Repo Stars" >}} {{< shields path="github/license/wechatsync/Wechatsync" alt="License: `GPL-3.0`" >}} {{< shields path="github/languages/top/wechatsync/Wechatsync" alt="Language: `Unknown`" >}}

![demo-picture-of-Wechatsync](https://static.osguider.com/subject/github/wechatsync/Wechatsync/28ae2406c5c0ce29aab32cbcbf10f615.png)

Wechatsync 是一个一键同步文章到多个内容平台的工具，支持今日头条、WordPress、知乎等主流自媒体平台，实现一次发布，多平台同步发布，从而提高个人生产力。

- 支持多种内容平台的无缝同步
- 内置 Markdown 编辑器，方便排版
- 基于浏览器插件，无需云端存储，保障安全性
- 提供网页正文提取功能，提高使用效率
- 兼容 WordPress XMLRPC 协议，可扩展其他网站支持

- 兼容微信公众号、知乎和稀土掘金等平台
- 提供在线使用功能，方便用户体验
- 拥有丰富的主题列表，可提交新主题以扩展示例文章
