OSHI 是一个基于 JNA 的免费 Java 操作系统和硬件信息库。

- 提供跨平台的系统信息检索，包括操作系统版本、进程、内存和 CPU 使用情况等。
- 无需安装额外的本地库，简化了使用过程。
- 支持多种平台，如 Windows、macOS 和 Linux 等。
- 能够获取详细的硬件信息，包括 CPU 负载、磁盘驱动器状态和网络接口统计数据。
- 包含示例代码，便于开发者快速上手。

## [grafana/grafana](https://github.com/grafana/grafana)

{{< shields path="github/stars/grafana/grafana" alt="Github Repo Stars" >}} {{< shields path="github/license/grafana/grafana" alt="License: `AGPL-3.0`" >}} {{< shields path="github/languages/top/grafana/grafana" alt="Language: `Unknown`" >}}

{{< github-opengraph user="grafana" repo="grafana" alt="cover" >}}

Grafana 是一个开放且可组合的可观察性和数据可视化平台。

- 支持从多个来源（如 Prometheus、Loki、Elasticsearch 等）可视化指标、日志和追踪。
- 提供快速灵活的客户端图形，具有多种选项进行数据展示。
- 可创建动态且可重用的仪表板，支持模板变量下拉选择。
- 允许通过临时查询和动态钻取探索数据，并并排比较不同时间范围的数据。
- 实现从指标到日志的无缝切换，并保留标签过滤器以便快速搜索日志。
- 可视化定义重要指标的警报规则，并向 Slack 等系统发送通知。
- 支持在同一图表中混合不同的数据源。
